Using the wreal Data Type

The AMS Designer simulator supports the Verilog®-AMS wreal data type, which lets you define a real number physical connection between structural entities.

wreal in; // Declares 'in' as a wreal net

A set of wreal sample libraries are installed under INST_DIR/tools/amsd/wrealSamples area. These libraries contain text models and also the versions that can be used with the current Xcelium and Spectre releases. The text models are directly accessible and usable in a command-line digital-centric flow.
