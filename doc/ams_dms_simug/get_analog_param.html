get_analog_param

Returns the value for the specified parameter on the Tcl command prompt.

This command is available only when the analog engine is enabled.

get_analog_param Command Syntax

get_analog_param
    -name paramName

get_analog_param Command Options

This section describes the options that you can use with the Tcl get_analog_param command.

-name

Specifies the name of the parameter whose value is to be displayed.

Example:

get_analog_param -name iabstol 

Displays the value of the parameter -iabstol.
