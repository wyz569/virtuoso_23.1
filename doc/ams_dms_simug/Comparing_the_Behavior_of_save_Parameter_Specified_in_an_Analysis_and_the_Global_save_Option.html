Comparing the Behavior of save Parameter Specified in an Analysis and the Global save Option

If you have the save parameter specified in an analysis statement (for example, tran, ac, and so on), the save parameter setting specified for that analysis overrides the save option specified using the global options statement. However, if save = none is specified in an analysis statement, then, the value specified using the global options statement is considered. The following table shows the behavior of the save parameter when specified in both transient analysis and global options statement.

tran Statement

options Statement

Behavior

save=none

save=none

Saves any random voltage signals.

save=selected

save=none

Saves only selected signals specified using the save statement in the netlist.

save=none

save=selected

Uses save=selected specified in the global options statement.

save=selected

save=selected

Saves only selected signals specified using the save statements in the netlist.
