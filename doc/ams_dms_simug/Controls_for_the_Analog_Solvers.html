Controls for the Analog Solvers

In the Spectre® AMS Designer simulator, the primary way of controlling the analog solvers is to define an analog simulation control file. The analog simulation control file is an ASCII text file containing commands that control the behavior of the analog solvers. It must not contain any subcircuit or model definitions or other Spectre/SPICE commands. The analog simulation control file must have .scs as the file extension in order to establish the Spectre language as the default language for the file. Commands in the analog simulation control file apply to xmsim (or to the simulation phase of xrun). 

The simulator lang statement specifies the language mode you want the software to use to evaluate succeeding statements.
