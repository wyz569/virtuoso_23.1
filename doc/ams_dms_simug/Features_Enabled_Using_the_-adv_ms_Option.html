Features Enabled Using the -adv_ms Option

You can now enable the following advanced mixed-signal verification functionalities using a single xrun command-line option, -adv_ms:

  • Connecting SystemVerilog Logic Ports to Electrical Using Wire/Interconnects
  • Connecting SystemVerilog Variable Logic Vector to SPICE Bus
  • Applying Bit-Wise Discipline for SPICE Bus Connections

The -adv_ms option is a sub-set of the -sv_ms option; all features documented in mentioned in this topic (under -adv_ms) can be enabled using the -sv_ms option, as well. See also, Features Enabled Using the -sv_ms Option.
