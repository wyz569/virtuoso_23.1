The spectre_compatible_version.xml File Location

To find the .amsd/spectre_compatible_version.xml file, use the following search order.

$CWD
$CDS_WORKAREA (user work area, if defined)
$CDS_SEARCHDIR (if defined)
$HOME
$CDS_PROJECT (project area, if defined)
$CDS_SITE (site setup, if defined)
xcelium_install_dir/share
