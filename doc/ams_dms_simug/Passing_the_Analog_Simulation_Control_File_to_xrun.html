Passing the Analog Simulation Control File to xrun

You can specify the analog control file directly on the xrun command line.

To pass the analog simulation control file to xrun:

  • Set the following command.

          xrun analogControlFileName fileName

          Example

          xrun adc.scs sar6bit.v

          Passes an analog control file called adc.scs to xrun to compile, elaborate, and simulate a design contained in a file called sar6bit.v.
