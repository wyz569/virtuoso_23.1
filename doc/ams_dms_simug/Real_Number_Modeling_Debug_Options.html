Real Number Modeling Debug Options

You can use the xrun command with the following options to debug issues in designs using real number modeling:

  • +wreal_res_info: Prints a report of all wreal nets in the design and their resolution functions.
  • -rnm_info: Prints a report of all .vams, wreal, SV-wreal nettypes, and SV-UDN structs within the design, including nettype declaration and coercion information of these nets.
  • -show_rnm_cover: Prints the file name and line number of RNM element inside a coverage group that enabled the DMSO license.

    Additionally, you can also use the -show_rnm_rand option to get the filename and line number of real randomization that enabled the DMSO license.
