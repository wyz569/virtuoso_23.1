SystemVerilog User Defined Nettype and Electrical Connections

You can connect a design that has SystemVerilog user-defined nettype (UDN) to electrical net connections by using the built-in nettype or custom UDN  to electrical bidirectional connect modules to set up a SystemVerilog Real Numbered Model UDN and a Verilog-AMS (or SPICE/Spectre) electrical port connected on a mixed-signal net design.

To set up a SystemVerilog UDN-Electrical connection, you can do the following:

  • Use the built-in nettype (EEnet) decalred in the EE_pkg.sv package file to connect electrical ports (Verilog-AMS or SPICE/Spectre). These connections are done by inserting built-in Connect Modules (EEnet_2_E.svams, E_2_EEnet.svams, EEnet_to_E_bidir.svams).
  • Create custom user-defined nettype (UDN) with resolution function and custom Connect Modules to insert between custom UDN and electrical connections.
