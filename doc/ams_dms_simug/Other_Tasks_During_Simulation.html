Other Tasks During Simulation

Some of the other tasks you can perform when you invoke the simulator are:

  • Updating Design Changes When You Run the Simulator: After editing a design unit, you can use xrun -r -update to recompile and re-elaborate all out-of-date design units before re-simulating.
  • Providing Interactive Commands from a File: If you want to load a file containing simulator commands (perhaps Tcl commands or aliases, or a key file containing commands saved from a previous simulation run), you can specify the input file with the -input option when you invoke the simulator. For example, xrun -r -input <filename>.
