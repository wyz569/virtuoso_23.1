Analog Simulation Control File

The analog simulation control file is an ASCII text file containing commands that control the behavior of the analog solvers. It must not contain any subcircuit or model definitions or other Spectre/SPICE commands. The analog simulation control file must have .scs as the file extension in order to establish the Spectre language as the default language for the file. Commands in the analog simulation control file apply to xmsim (or to the simulation phase of xrun).
