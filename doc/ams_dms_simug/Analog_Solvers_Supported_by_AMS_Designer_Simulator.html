Analog Solvers Supported by AMS Designer Simulator

The figure below shows the analog solvers that are supported with the AMS Designer simulator:

The following table shows how to enable the analog solvers in the AMS Designer simulator:

Analog Solver

How to enable...

Spectre (default)

xrun amscf.scs

APS

xrun -spectre_args "+aps" amscf.scs

Fast APS

xrun -spectre_args "++aps" amscf.scs
Spectre X xrun -spectre_args "+preset=cx|ax|mx|lx|vx" amscf.scs

XPS (MS mode)

By default, APS is enabled for the analog partition.

xrun -spectre_args "+preset=cx|ax|mx|lx|vx" amscf.scs

XPS (MS mode)

With fast APS enabled for the analog partition.

xrun -spectre_args "+ms +preset=cx|ax|mx|lx|vx" amscf.scs
