Using Wildcard‐Named Port Connections in SV Bind Statements

Wildcard‐named port connections represented using the dot‐star notation (.*), is supported in the SV bind statements connecting to Verilog‐AMS port connections. To enable the dot‐star notation functionality, use the xrun command‐line option, -sv_ms.

The following is an example of using the dot‐star notation in a Verilog‐AMS port connection:

% xrun -sv_ms amscf.scs vtop.vams top.sv inc.sv

//top.sv
module top();
vtop x1(clkin); <-- VAMS
bind vtop inc xinc(.*); <--- Target is VAMS
endmodule

//amscf.scs
amsd {
    ie vsup=1.8
}
//vtop.vams
module vtop();
electrical a;
wreal b;
...
endmodule

//inc.sv
module inc(a, b);
inout a;
inout b;
...
endmodule
