License Checkout Order for Mixed-Signal Designs

To run AMS features in Xcelium, you must have access to a corresponding license or combination of licenses. You can change the license checkout order for the Spectre simulators by using the -spectre_args +lorder option.

To change the checkout order of Spectre AMS licenses used in the Xcelium simulator, use the -ams_lorder option. You can use the following arguments with the -ams_lorder option:

  • SAMSD: Checks out the Spectre AMS Designer license.

  • SAMSC: Checks out the Spectre AMS Connector license.

  • MMSIM2: Checks out the MMSIM2 license. In this case, the MMSIM2 license has higher priority than licenses with one MMSIM.

  • MMSIM: Checks out only the MMSIM license. In this case, licenses with one MMSIM have higher priority than 2MMSIM. 

  • XCELIUM: Check out only the Xcelium-based licensing for digital+mixed-signal simulations, with no analog solver, specify -ams_lorder XCELIUM. You cannot use the other four checkout order arguments along with the XCELIUM license order.

To check out the license in the order MMSIM, SAMSD, and SAMSC, specify -ams_lorder MMSIM:SAMSD:SAMSC.

AMS features in Xcelium can be accessed using the product a-la-carte licenses (Spectre AMS Connector or Spectre AMS Designer) or MMSIM tokens. If both are available, product a-la-carte licenses are checked out first.

Additionally, to specify the license checkout order of Xcelium simulator with the mixed-signal option, you can use the -license_order command. You can use the following arguments:

  • XCELIUM_SC_ONLY: Checks out only Xcelium Single Core as the main Xcelium license. 

    For mixed-signal simulations with advanced MSV and Xcelium centric feature set requiring no analog solver, xrun checks out Xcelium Single Core and Xcelium MixedSignal App.

    For mixed-signal simulations with advanced MSV and Xcelium centric feature set requiring an analog solver, xrun checks out Xcelium Single Core, Xcelium MixedSignal App, Spectre AMSD Connector (SAMSD).
  • XCELIUM_SC_DMS: Checks out only Xcelium Single Core and Xcelium Mixed-Signal App licenses for simulation.

For mixed simulation, xrun checks out only Xcelium Single Core and Xcelium Mixed-Signal App, when analog solver is not used. And, xrun does not check out SAMSD or 2MMSIM to replace Xcelium Single Core, when analog solver is used.

For pure digital designs without the Mixed-Signal App feature, this option behaves the same as -license_order XCELIUM_SC_ONLY.
