Universal Connect Module (UCM)

The Universal Connect Module (UCM) consolidates and replaces the existing connect modules in connectLib in the Xcelium installation. It is enabled by default in AMS Spectre FX mode. To enable UCM in APS and Spectre X mode, use the xrun option -ams_ucm.

The UCM supports the basic, full_fast, and full features in one set of connect modules with connect rule uCR:

For electrical-logic

For electrical-real

For logic-real

uE2L

uE2R

R2L

uL2E

uR2E

L2R

uBidir

uERBidir

RL_bidir


The UCM supports the following constant and dynamic power supply features in one set of connect modules.

  • Constant supply (vsup)
  • Dynamic supply (vddnet, vssnet)
  • Inherited connection supply
  • Supply sensitive supply
  • LPS
