Specifying a Host for Spectre

 To run Spectre on a local host:

  • Use the xrun command with the -analogsolver command-line option.
    Example:
    xrun -analogsolver "spectre"
    AMS Designer runs the spectre executable on the local host. This is the default behavior. Alternatively, you can use a script or a resource scheduler, provided it boots Spectre.

To run Spectre on a remote host:

  • Use the bsub command.
    xrun -analogsolver "bsub -R \"OSNAME==Linux &7 OSREL==EE70 && SFIPLATFORM==c0700\" -n 1 -q lnx64 -J spectre"
