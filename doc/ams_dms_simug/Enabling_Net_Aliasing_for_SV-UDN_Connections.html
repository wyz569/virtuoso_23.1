Enabling Net Aliasing for SV-UDN Connections

Use the system function $real_net_alias() to alias a local net to a hierarchical net through a string reference within connect modules and regular modules.

In SV-UDN real net array connections, both the real_net_reference and hierarchical_reference_string arguments must be of the same nettype. And, the $real_net_alias system function does not support SV-UDN nets with struct type; and, returns zero value in this case.
