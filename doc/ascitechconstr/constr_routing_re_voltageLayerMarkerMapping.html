voltageLayerMarkerMapping

spacings(
( voltageLayerMarkerMapping tx_layer)
 'layer t_layer 
 'purpose t_purpose 
) ;spacings

(Virtuoso Advanced Node for Layout Standard) Specifies the LPP to be used for marking nets that transition in phase. The constraint applies on the layer on which the voltage is measured.

Values

tx_layer

The layer on which the voltage is measured.

Type: String (layer and purpose names) or Integer (layer number)

Parameters

'layer t_layer

The layer to use for the marking shape.

'purpose t_purpose

The purpose to use for the marking shape.

Example

spacings(
    ( voltageLayerMarkerMapping "Metal1" 
     'layer "Metal1" 
  'purpose "m1min" 
 )
) ;spacings
