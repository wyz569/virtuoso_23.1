deltaVLayerMarkerMapping

spacings(
( deltaVLayerMarkerMapping tx_layer
 'layer tx_markerLayer 
 'purpose tx_markerPurpose 
) 
) ;spacings

(Virtuoso Layout Suite EXL and higher tiers) This constraint is used to indicate that a marker shape is generated to overlap two shapes on different nets on the constraint layer to record the maximum delta voltage between them. The constraint specifies the layer and purpose of the marker shape generated.

Value

tx_layer

The layer on which the two different net shapes are found.

Type: String (layer and purpose names) or Integer (layer number)

Parameters

'layer tx_markerLayer

  

The layer on which the marker shape overlapping the two net shapes are to be generated.

Type: String (layer and purpose names) or Integer (layer number)

'purpose tx_markerPurpose

The purpose on which the marker shape overlapping the two net shapes are to be generated.

Type: String (purpose name)
