minCornerVoltageSpacing

spacings(
( minCornerVoltageSpacing tx_layer
 'prlRange f_prlRange
 'deltaVoltage f_voltage
 ['subtractTrim]
 ['endOfLineWidth f_eolWidth]
 f_minSpacing
)
) ;spacings

(Virtuoso Advanced Node for Layout Only) Specifies the minimum spacing required based on the voltage between the corners of shapes on a specified layer.

Values

tx_layer

The layer on which the constraint is applied.

Type: String (layer and purpose names) or Integer (layer number)

f_minSpacing

The minimum spacing required.

Parameters

'prlRange f_prlRange

  

The constraint applies only if the parallel run length between the two shapes falls in this range.

Type: Floating-point values specifying a range of parallel run lengths that trigger the constraint

'deltaVoltage f_voltage

The constraint applies only if the maximum voltage difference between the two shapes is greater than this value.

'subtractTrim

If true, and the layer is trimmable, the trim will be subtracted from the layer before checking is performed.

Type: Boolean

'endOfLineWidth f_eolWidth

(Virtuoso Layout Suite EXL and higher tiers) The constraint applies only to end-of-line edges with width less than or equal to this value.

 
Examples
spacings(
( minCornerVoltageSpacing "Metal1"
    'prlRange "[-0.06 0]"
    'subtractTrim
    'deltaVoltage 0.12  0.22
)
) ;spacings
