minEndOfLineAdjExtension

orderedSpacings(
( minEndOfLineAdjExtension tx_metalLayer tx_cutLayer
 'eolSpacing f_eolSpacing
 'width f_width
 ['orthogonalPair]
f_extension 
)
) ;orderedSpacings

(Virtuoso Advanced Node for Layout Only) Specifies a cut extension when an adjacent shape within a specified distance is present.

Values

tx_metalLayer tx_cutLayer

The layers on which the constraint is applied.

Type: String (layer and purpose names) or Integer (layer number)

f_extension

The minimum required extension value.

Parameters

'eolSpacing f_eolSpacing

The constraint applies when the adjacent shape is within this distance from the end-of-line.

'width f_width

The constraint applies to shapes with an end-of-line less than or equal to this value.

'orthogonalPair

The constraint applies to orthogonal pairs of shapes.

Example

This example specifies the required extension of M1 over V1 to be 0.45 when another M1 shape is present within 0.7 of the end-of-line edge in the orthogonal routing direction when the enclosing shape has a width of less than or equal to 0.5.
