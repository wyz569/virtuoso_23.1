deltaVLabelMapping

spacings(
( deltaVLabelMapping tx_layer
 'layer tx_labelLayer 
 'purpose tx_labelPurpose 
) 
) ;spacings

(Virtuoso Layout Suite EXL and higher tiers) Specifies the layer and purpose of the deltaV label generated to represent the maximum voltage delta between two shapes on different nets. The deltaV label overlaps the deltaV marker shape on the constraints layer generated by using the deltaVLayerMarkerMapping or deltaVLayerPairMarkerMapping constraints. This marker overlaps the two net shapes.

Value

tx_layer

The layer on which the deltaV marker is generated.

Type: String (layer and purpose names) or Integer (layer number)

Parameters

'layer tx_labelLayer

  

The layer on which the deltaV label is generated.

Type: String (layer and purpose names) or Integer (layer number)

'purpose tx_labelPurpose

The purpose on which the deltaV label is generated.

Type: String (purpose name)
