Virtuoso Technology Data Constraints Reference
Product Version IC23.1, November 2023

minVoltageSpacing (One layer)

spacingTables(
( minVoltageSpacing tx_layer
 (( "voltage" nil nil ))
 ['horizontal | 'vertical]
 (g_table)
)
) ;spacingTables

Specifies the minimum spacing between two wires on a layer when the maximum difference in voltage between the two wires is greater than or equal to the specified voltage. The difference in voltage is calculated as follows:

Vdiff = max(Vmax of each shape) - min(Vmin of each shape)

Layer-purpose pairs are not supported for this rule. If a layer-purpose pair is found, a warning is displayed and the constraint is created as a non-purpose-aware constraint, that is, the purpose is reassigned as techcNoPurpose.

Shape1 to Shape2

Shape2 to Shape3

Voltage difference

max(5,2) - min(0,1) = 5-0 = 5

max(2,3) - min(1,0) = 3-0 = 3

Values

tx_layer

The layer on which the constraint is applied.

Type: String (layer name) or Integer (layer number)

"voltage" nil nil

This identifies the index for table.

g_table

The format of the table row is as follows:

(f_voltage f_distance)

where,

  • f_voltage is the voltage difference between two wires. The voltage difference must be greater than or equal to this value.
  • f_distance is the minimum required spacing.

Type: A 1-D table specifying floating-point voltage and spacing values

Parameters

'horizontal | 'vertical

  

(Virtuoso Advanced Node for Layout Only) The direction in which spacing is measured. If the direction is not specified, spacing is measured in any direction.

Example

The spacing between two Metal1 wires must be at least:

  • 0.065 if the maximum voltage difference between the wires is greater than or equal to 0.0V and less than 1.5V
  • 0.085 if the maximum voltage difference between the wires is greater than or equal to 1.5V and less than 3.3V
  • 0.105 if the maximum voltage difference between the wires is greater than or equal to 3.3V

It must be measured in the vertical direction.

spacingTables(
    ( minVoltageSpacing "Metal1"
     (( "voltage" nil nil ))
  (
    0.0  0.065
    1.5  0.085
    3.3  0.105
  )
  'vertical
 )
) ;spacingTables
