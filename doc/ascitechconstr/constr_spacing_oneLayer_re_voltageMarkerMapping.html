voltageMarkerMapping

spacings(
( voltageMarkerMapping tx_layer
 'layer t_layer 
 'purpose t_purpose 
 f_voltage
)
) ;spacings

Specifies the layer and purpose for which voltage markers will be created.

Values

tx_layer

The voltage marker is created on a shape on this layer on the net that is being labeled.

f_voltage

A marker is created for nets with the specified voltage.

Parameters

'layer t_layer

The layer on which the marker will be created.

'purpose t_purpose

The purpose on which the marker will be created.

Example

Markers showing the specified voltages will be drawn on nets on the layer-purpose pair Metal1 m1min. 

spacings(
    ( voltageMarkerMapping "Metal1" 
     'layer "Metal1" 
  'purpose "m1min" 
  1.2
 )
) ;spacings
