integrationColorModel

colorControl(
( integrationColorModel "locked" | "any" 
)
) ;colorControl

Specifies that instances of blocks bound to a technology or instances of a specific block can have color shifting applied to them.

colorControl is similar to other constraint categories such as spacings, orderedSpacings, and routingGrids.

Values

"locked" | "any"

The value any signifies that any kind of color shifting can be applied. This is the default value that also applies if the contraint is not specified.

The value locked signifies that coloring applications should not apply color shifting.Type: String (layer and purpose names) or Integer (layer number)

Parameters

None
