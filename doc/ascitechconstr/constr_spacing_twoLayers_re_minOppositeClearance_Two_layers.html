minOppositeClearance (Two layers)

orderedSpacings(
( minOppositeClearance tx_cut1 tx_cut2 
 'prl g_prl
 g_spacing 
)
) ;orderedSpacings

(Virtuoso Advanced Node for Layout Only) Specifies the cut spacing between a cut of any cut class on a layer and two cuts of any cut class on a second layer.

Values

tx_cut1

The first cut layer on which the constraint is applied.

Type: String (layer and purpose names) or Integer (layer number)

tx_cut2

The second cut layer on which the constraint is applied.

Type: String (layer and purpose names) or Integer (layer number)

g_spacing

The minimum spacing required between the via cuts.

Parameter

'prl g_prl

The constraint applies only if two cuts on tx_cut2 individually have a parallel run length greater than or equal to this value to tx_cut1.
