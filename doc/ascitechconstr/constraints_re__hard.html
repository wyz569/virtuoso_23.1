'hard

'hard

Identifies the constraint as a hard constraint that must be satisfied. This parameter can be specified for any constraint.

Example

orderedSpacings(
    ( minOppExtension "poly1" "metal2" (2.1 0.7) 'hard )
) ;orderedSpacings
