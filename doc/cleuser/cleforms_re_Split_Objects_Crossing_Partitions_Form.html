Split Objects Crossing Partitions Form

Splits the crossing objects across hierarchical design partitions.

This form is displayed when the top design partition contains hierarchical design partitions. .

Field Description

Top design

Specifies the name of the top design that is initialized for Concurrent Layout editing.

Hierarchical designs

Shows the number of hierarchical subcells included in the top design partition. You can select the check boxes listed below to specify the hierarchical subcells for which you want to split the objects crossing design partitions.
