Preview Options

Additional preview options are displayed when you right-click the Preview option on the title bar of the Design Management pane.

The following table lists the functions of commands on the right-click menu of the Preview option:

Command

Description

Highlight Modified Figures

Highlights the updated figures in the selected design partition.

Default value: Selected

Dim Display of Unmodified Figures

Dims the color of figures that have not been modified.

Default value: Deselected

Hide Original Figures of Changes Not Imported

Hides the figures for which changes have not been imported.

Default value: Selected
