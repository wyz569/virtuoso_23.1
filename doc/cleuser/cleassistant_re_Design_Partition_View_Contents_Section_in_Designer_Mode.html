Design Partition View Contents Section in Designer Mode

The Design Partition View Contents section becomes available when you are editing a design partition view.

This section comprises three main components:

  • Design Partition View Toolbar
  • Summary Pane
  • Details Pane
  • Information Pane

Each of these components is described in detail below.

Design Partition View Toolbar

The following table lists the functions of the different options on the Design Partition View toolbar:

Icon

Command

Description

Zoom to Selected

Zooms to the selected object in the summary pane.

Auto Zoom

Zooms automatically into selected design partition or selected changes.

Previous Node

Lets you move to the previous entry in the Details pane.

Next Node

Lets you move to the next entry in the Details pane.

Cross Highlight in Canvas

Toggles the display of halo over the edited object in the canvas.

When enabled, the selected objects in the Details pane are haloed in the canvas

Select All

Selects all objects in the Details pane.

Deselect All

Deselects all objects in the Details pane.

Cross Select in the Canvas

Toggles the cross selection of objects in Details pane to the canvas

Select Inside Partition

Toggles between Off and Only Select Inside Partition modes.

Off

Toggles between Off and Only Select Inside Partition modes.

When set to Off, there are no restrictions about creating or editing in the current design partition.

Only Select Inside Partition

Prevents selecting objects that are fully outside and area-based design partition. Objects that are fully or partially inside or have been temporarily moved outside the current partition for editing are also selectable. This option is selected by default.

In layer-based design partitions, the CLE filter is selected in the Layer Palette and only the valid layers are selectable. However there is no restriction on vias because the Create Via, Create Wire, and routing commands can use all vias. You need to sign off the out of partition warnings in the assistant.

Only Edit Inside Partition

Lets you edit only within the current design partition.

Supported commands are: Create – Shapes, Create – Instance, Create – Pin, Create – Label, Create – Fluid Guard Ring, Create – Wiring – Wire, Create – Via, Edit – Move, Edit – Copy, Edit – Stretch, Edit – Delete, Edit – Quick Align, Edit – Advanced – Reshape.

In a layer-based design partitions, only valid layers and valid vias are selectable

Change Selection Filters on Nets

Selects only those shapes that are unassigned or assigned to a net in the selection filter on nets, or have been modified.

Pcell is selectable if at least one instance terminal is unassigned or assigned to one net in the net filter. Regular instances are always selectable to enable the user to Edit In Place (EIP) or Descend Edit. For a net-based design partition, the default filter is the net set.

Show Signed-Off Changes

Show hidden signed off checks in the Details pane.

Hide Signed-Off Changes

Hide signed off checks in the Details pane. The Summary pane shows a number, which are in the format visible or total, if there are hidden signed-off checks.

Find Object From Canvas

Finds an object in assistant by clicking an object in the canvas without altering the canvas selection.

Remove Changes from Design Partition View

Displays the Remove Changes from Design Partition View Form form that lets you remove the changes for which the filter state has been set from the current design partition view. You can set the filter state of a change from the Filter State option in the Details Pane.

Search

Type the search keyword to filter the result in the Details pane. Use the drop-down menu to customize your search.
