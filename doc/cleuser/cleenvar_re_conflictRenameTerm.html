conflictRenameTerm

layout.cle   conflictRenameTerm boolean { t | nil }

Description

Specifies whether conflicting terminals should be renamed during merge. The default is t, which means that conflicting terminals are renamed.

GUI Equivalent

Command:

Concurrent – Options

Field:

Term

Examples

envGetVal("layout.cle" "conflictRenameTerm")
envSetVal("layout.cle" "conflictRenameTerm" 'boolean t)
envSetVal("layout.cle" "conflictRenameTerm" 'boolean nil)
