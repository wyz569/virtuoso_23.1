Design Partition Options in Designer Mode

The following design partition options are available:

Command

Description

Return to Top

Saves the edits made and returns to the top design.
This option is available after you choose Edit in Design Partition from the top design.

View Instance/Clone Occurrence

Displays the View Instance Occurrence Form or View Clone Occurrence Form that lets you view occurrences of an instance or a clone in the design hierarchy.

View Design Partition Definition

Displays the View Design Partition Form that lets you check design partition definitions.

Return to Design Partition

Returns to edit in the figure group of an object-based design partition. The option is available when you return from the figure group of the design partition to the top level.

You can also click on the blue Alert icon to return.
This option is available only for object-based design partitions.

Show Detailed Changes

Highlights the updated information in the design.

Import Peer Design Partition

Displays the Import Peer Design Partition Form form that lets you import updates made by a peer designer in your Design Partition view.

Post-Edit Check All Edits

Touches all changed objects to allow the interactive checkers, such as post-edit DRD or connectivity extraction, to verify these edits.

This option is not available after you have imported peer partitions.

Check Edit Conflicts

Displays the Check Edit Conflicts Form form that lets you view a log file that lists any edit conflicts that might have occurred due to changes made in the selected design partition.

Save Figures for Comparison

Displays the Save Figures for Comparison Form form that lets you specify the views that you want to save and compare the updates made.

Dump Pegasus/PVS Window Rule for DRC

Generates the window rule file to restrict the batch Pegasus or PVS to check only within the design partition area.

XStream Out from VM for DRC

Translates the current design partition view to a stream file from virtual memory for running DRC checks.

XOasis Out from VM for DRC

Translates the current design partition view to an OASIS file from virtual memory for running DRC checks.

Add Design Partition Nets Set

Switches to the Navigator assistant and displays the list of nets crossing or inside the current design partition under the CONCURRENT LAYOUT category.

Remove Temporary Objects

Deletes temporary objects, such as virtual pins created by Virtuoso Space-based Router (VSR) for routing inside area boundaries of the design partition.

Generate Temporary Pins

Generates temporary pins at the area boundaries of the design partitions.

Select Temporary Pins

Selects all temporary pins for deleting or cross-selecting their nets.

Make Opaque Instance

Displays the Make Opaque Form that lets you convert instance selected in the design to opaques instances.

Revert Opaque Instance

Displays the Revert Opaque Instances Form that lets you convert selected opaque instances into normal instances.
