Rejecting a Submitted Design Partition

To reject merge requests:

  1. Click Merge/Unmerge in the Concurrent Layout assistant or choose Concurrent – Merge.
    The Merge form is displayed.
    In the Concurrent Layout assistant, the design partition view status changes to Committed.
  2. Select Reject in the Action field for the design partitions you want to reject.
    If the design partition status is Merging, undo the changes until the status changes back to Submitted and then reject it.
  3. Click Apply.
    The Reject Submission form is displayed.

  4. Specify a reason for rejecting the merge request for the rejected design partitions and click OK.
    In the Concurrent Layout assistant, the status of the rejected design partitions changes to Rejected and the status of the merged partitions changes to Committed.
