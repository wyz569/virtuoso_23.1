Save as Design Partition View Form

Saves updates only for the specified design partition view.

The option to open this form is available only when you open the cellview in full design view. Full design view of a design partition can be saved using the Save As Full Partition View form.

Field

Description

Library

Specifies the name of library of the design partition.

Cell

Specifies the top cell name of the design partition.

View

Specifies the design partition view name of the design partition.
