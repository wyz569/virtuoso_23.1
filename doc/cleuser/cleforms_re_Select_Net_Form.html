Select Net Form

Lets you select the nets that are attached to the selected design partition in the Navigator.

Field Description

Design Partition

Lets you select a design partition.

Sort

Sorts the nets in the Ascending or Descending order.

List of nets lets you select or deselect the listed nets.

Select All

Selects all nets.

Deselect All

Deselects all nets.
