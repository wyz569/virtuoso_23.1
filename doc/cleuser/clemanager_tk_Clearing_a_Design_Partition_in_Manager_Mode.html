Clearing a Design Partition in Manager Mode

After all design partitions are merged with the top design, you might want to delete all design partitions and exit the Concurrent Layout Editing environment. To clear all design partitions:

  1. Choose Clear All Design Partitions from the Merge dropdown list in the Concurrent Layout assistant or choose Concurrent – Clear All Design Partitions.
    The Clear All Design Partitions form is displayed.
  2. Select the Keep design partition definitions check box if you want to retain design partition definitions,
  3. Select the Keep design partition areas check box in the area-based design partitions, if you want to retain the design partition areas created using Define Design Partition form.
  4. Click OK.
    All design partitions will be deleted and you can edit the design without using Concurrent Layout.
In a DM environment, users usually do not have permissions to remove files from the DM vault, therefore clearing design partitions might just retire them, and the information might still exist in the DM vault. To automatically delete design partitions in DM environment using the Clear Design Partitions command, set the autoDeletePartitionView environment variable to t.
