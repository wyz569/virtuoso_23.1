Revert Opaque Instances Form

Reverts the selected opaque instances into normal instances.

Field

Description

Total

Specifies the total number of opaque instances in the selected cell and the number of currently selected instances in the form.

Filters the instances for the selected cell.

Name

Name of the instance.

Library

Name of library for the instance.

Cell

Cell name for the instance.

View

View name of for the instance.

Nets to extract

Name of nets associated with the instance.

Level to extract

Depth level of the instance.

Cross Highlight in the Canvas

Instances selected in the form are highlighted in the design.

Cross Select in the Canvas

Instances selected in the form are selected in the design.

Zoom to Selected Instance

Zooms to the instance currently selected in the form.
