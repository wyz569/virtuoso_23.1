Known Problems in the Pegasus Interactive Release

This document describes Pegasus Interactive known problems in the Pegasus 23.11 release, and where possible, provides solutions, or possible workaround, for these problems. This is not a complete list of the known problems and solutions; however, this document will be updated periodically.

Contact your Cadence representative for the most recent KP&S issues.

CCR 1344216: Using 'old' style of LSW layer window result to incorrect Pegasus Interactive out for Any Visible preset

Problem: Pegasus Interactive Dynamic Rule Filtering: Use Layer Visibility mode is compatible with Palette Assistants layer display selector.

A previous method of layer selection based on LSW window (which is activated by setup of environment variable iCDS_PALETTE_OFF) is not supported by Pegasus Interactive.

Solution: For information only.
