Show Registers Form

Use the Show Registers form to display the waveforms produced during a simulation in register format.

Show Registers is supported only with icfb and icds workbenches.

The following table describes the fields available in the Show Registers form.

Field Description

Simulation Run Directory

Specifies the name and path of the current simulation run directory as a read-only field.

To change the directory, click Cancel and select the Initialize command again.

Waveform File Name

Specifies the name of the waveform file to display.

The system expects this file to be within /run_dir/raw. The default waveform file is waves. Most of the standard interfaces do not require that you change this field. If you are using a tool that produces multiple waveform files, such as the Cadence HSPICE interface, you need to change this field to look at the waveforms for each analysis run.
