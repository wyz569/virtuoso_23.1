Show Waveforms Form

Lets you display the waveforms produced during a simulation.

The following table describes the fields available in the Show Waveforms form.

Field Description

Simulation Run Directory

Specifies the name and path of the current simulation run directory as a read-only field.

To change the directory, click Cancel and select the Initialize command again.

Waveform File Name

Specifies the name of the waveform file to display.

The system expects this file to be in the raw directory in the simulation run directory. The default waveform file is waves. Most of the standard interfaces do not require that you change this field. If you are using a tool that produces multiple waveform files, such as the Cadence HSPICE interface, you need to change this field to look at the waveforms for each analysis run.
