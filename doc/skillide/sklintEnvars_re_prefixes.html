prefixes

sklint.Input prefixes string "prefixes"

Description

Specifies the value for input argument ?prefixes, which lists the acceptable package prefixes for functions and global variables.

Default is empty string.

GUI Equivalent

Command:

Options – Lint

Field:

Package Prefixes

Examples

envGetVal("sklint.Input" "prefixes")
envSetVal("sklint.Input" "prefixes" 'string "pre_")
