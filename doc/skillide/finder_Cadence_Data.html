Cadence Data

You can find Cadence-supplied information here:

your_install_dir/doc/finder/language/functionArea/*.fnd
Database files must have .fnd as the file extension, such as chap1.fnd. All files must contain information in the appropriate Data Format.

where

your_install_dir

is the directory where your Cadence software is installed.

Type cdsGetInstPath and press Return on the command line in your Command Interpreter Window to find your_install_dir.

language

is the language type, such as SKILL.

functionArea

is a descriptive subdirectory name for the product information, such as SKILL_Language or SKILL_Development.

Underscores used in subdirectory names to separate words appear as spaces in the Searching cyclic field. The word Only appears after each entry.
