Troubleshoot Extract Issues

You can face the following issues while using the Extract step in the abstract generation process.

ABS-13006

Message:

Cell name: layer 'POLY' is not listed in the Layer Assignment for name Extraction table. This layer is required for antenna extraction. Select 'POLY' in the Layer column and specify other information for this layer to be extracted. See Extracting Power Nets in Standalone Abstract Generator and Specifying Layers and Geometries for Extraction.

ABS-13007

Message:

Cell name: layer 'DIFF', which is required for antenna extraction, is not listed in the Layer Assignment for name Extraction table. For antenna extraction, select 'DIFF' in the Layer column and specify the layer 'DIFF' as type '(DIFF andnot POLY)', where DIFF and POLY are the diffusion and polysilicon layers in question. See Specifying Layers and Geometries for Extraction and Specifying Layers and Geometries for Extraction.

ABS-13008

Message:

Cell name: either the layer 'DIFF' is not specified as type '(diff diff andnot poly)' in the Layer Assignment for name Extraction table or the geometry specification for a layer contains two or more layers from the Layer column. Select 'DIFF' in the Layer column and specify it as type '(diff diff andnot poly)', where diff and poly are to be replaced with the real names of the diffusion and polysilicon layers in question. If a geometry specification uses a temporary layer to limit extraction and the temporary layer is a combination of two or more layers from the Layer column, do not select the individual layers in the Layer column.

Description:

This message appears when a diffusion layer is not specified in one of the layer assignment for extraction tables. This message might also appear when a geometry specification contains two or more layers from the Layer column. See Extracting Power Nets in Standalone Abstract Generator and Layer Assignment Tables.

ABS-13012

Message:

Cell name: a possible must join terminal name has been ignored because the terminal was not listed in the Create Must Connect Pins option on the name tab. Ensure that all must connects are joined for a specific terminal, either by activating the Always option on the Signal tab or listing the terminals in the Only on terminals named field.

ABS-13014

Message:

Cell name: there is no pin geometry on the net name. To create pins on a specific layer, select the Create Pins check box corresponding to the layer in the Layer Assignment for name Extraction table.

Description:

It is usually unnecessary to create pins on polysilicon, contact, or diffusion layers. See Specifying Layers and Geometries for Extraction.

ABS-13018

Message:

Cell name: net name process antenna calculations for terminal name on instance name have not been used.

Description:

Abstract Generator has failed to identify any extracted nodes as representing instance terminal pin shapes. Because the nodes have not been matched to an instance pin, the associated antenna calculations have not been taken into account. This problem could be a result of the net not being fully routed in DEF.

ABS-13019

Message:

Cell name: net name process antenna type value has been ignored because it is different from other layerName values for terminal name on instance name. This might indicate that the antenna information stored on a terminal is inconsistent with itself. This could occur if there is an error in the antenna calculation or the property values have changed in the abstract. The inconsistency might also indicate that the pin area for the instance terminal has changed since it was first subtracted in the Adjust step.

ABS-13021

Message:

Cell name: a short has been detected between the nets netName and netName during antenna calculation. This indicates a problem in the design unless the nets are connected by a polysilicon resistor. Though the short, if it exists between the nets, is ignored for antenna calculation but is considered for the signal net extraction because either the Use different layer assignments for antenna calculations only option is selected or the extraction depth has been limited.

Troubleshooting:

When you are performing antenna extraction in analog blocks, specify polysilicon andnot resistor in the Layer Assignment for Signal Extraction and Layer Assignment for Power Extraction tables. Remove the resistor part in the Layer Assignment for Antenna Extraction table.

ABS-13022

Message:

Cell name: no pin shape will be created on the layer name because the layer does not have net purpose defined in the technology data. Ensure that the technology file is correct. Otherwise, check the design.

Troubleshooting:

Use the Technology File Manager to define the missing LPPs in the techLayerPurposePriorities section of the technology file.
