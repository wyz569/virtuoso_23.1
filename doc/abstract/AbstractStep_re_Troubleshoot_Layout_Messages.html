Troubleshoot Layout Messages

ABS-11804

You have selected one of the extraction options (signal, power, or antenna extraction) and there are shapes on purpose boundary in the layout view for at least one of the layers entered in an extract geometry specification (GSpec) table. For some designs, you do not need to take any action. However, if your design contains large shapes on purpose boundary, this can cause extraction problems (such as shorts in a design) which are only discovered after a protracted extraction process.

You have chosen to perform either signal, power, or antenna extraction, but there are shapes on purpose boundary in your layout view. Large boundary shapes can cause extraction problems, such as shorts, but can be remedied by excluding the purpose boundary by using General options dialog box or by entering the following (example) geometry specification: metal1 -> (metal1 drawing) or (metal1 pin) or (metal1 net) in the appropriate GSpec location in the Extract step.

Troubleshooting: Remove purpose boundary from the geometry specification. For example, instead of specifying metal1, type the following instead.

(metal1 drawing) or (metal1 pin) or (metal1 net)

Even if you do not perform extraction, the pins created in the Abstract step could still be affected if you have blockage shapes on the layout.

Suppress Messages Issued by Abstract Generator

Use the suppressMessage option to suppress INFO and WARNING messages issued by Abstract Generator. 

absSetOption(suppressMessage <id1> <id2> <id3> …)

This option suppresses the messages with the mentioned message IDs. ERROR messages are not suppressed. If any ERROR message ID is mentioned in suppressMessage, then a warning is issued stating that this is an ERROR message and will not be suppressed.

For every suppressed message, an INFO message is displayed that this message is suppressed and will not be shown. This INFO message is displayed only once. At the end of the Abstract step, a table shows the messages that have been suppressed in the current run, and this information is displayed for every cell.
