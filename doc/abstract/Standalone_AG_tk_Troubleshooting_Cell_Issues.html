Troubleshoot Cell Issues

This section lists the common problems found in cells in the Virtuoso Studio design environment and provides information about how to fix these to enable Abstract Generator to run.

ABS-11001

Cell name: The view name view has been removed.

Cellviews are created upon the successful completion of each flow step in the abstract generation process. This message is generated when a cell view has been removed outside of Abstract Generator.

Troubleshooting: You can recreate the missing views by importing either GDSII (File – Import – Stream (GDSII)) or DEF data (File – Import – DEF). Alternatively, you can run the appropriate abstract generation steps.

ABS-11004

Cell name: Failed to open the view name view because message.

This error is issued when you try to open a non valid cell view.

Troubleshooting: Check the cell and remove from it the library if required.

ABS-11011

Cell name: The direction of the name terminal has changed from direction to direction. This change will invalidate any abstract generation steps already run. The terminal directions can be modified in the Terminal Properties form.

ABS-11012

Cell name: The type of the name terminal has changed from termType to termType. This change will invalidate any abstract generation steps already run. The terminal type can be modified in the Terminal Properties form.

Troubleshooting: If this change was not intentional, use the Terminal Properties form to restore the terminal type to its original setting.

ABS-11013

Cell name: The shape of the name terminal has changed from shape to shape. This change will invalidate any abstract generation steps already run. The terminal shape can be modified in the Terminal Properties form.

Troubleshooting: If this change was not intentional, use the Terminal Properties form to restore the terminal type to its original setting.

ABS-11024

Failed to process the cell because the step name view reason. Check the cell directory and close all applications that is using the same step and then try again.

Troubleshooting: Check the cell directory and remove any locked files or close an application that is using the same flow step.
