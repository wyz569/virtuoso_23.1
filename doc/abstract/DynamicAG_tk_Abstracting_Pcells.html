Abstracting Pcells

To generate an abstract of a Pcell in the layout window:

  1. Choose Tools – Dynamic Abstract Generation For Pcells – Abstract Pcells.
    The Abstract Pcells form is displayed as shown in the figure below. When you point to an option in the Abstract Pcells form, an info balloon is displayed that describes the purpose of the option.
  2. In the Selection section, choose All Pcells to create abstracts for all the Pcells in the design. If you do not want to create the abstracts for all the Pcells, you can select the required Pcells in the layout canvas or from the Navigator assistant and ensure that the Selected Pcells option is selected. After you have run the command, you can select additional Pcells for abstraction, if required, and run the command again.
    The abstracts are created for all Pcells that are selected, except the ones that are unmarked for abstraction.
  3. Check whether the abstracts were created successfully using one of the following methods:
    • View the blockages created at level-1in the layout canvas, as shown in the figure below.
    • View the status icons associated with the various Pcells in the Navigator assistant.
  4. In Options – Blockages section, you can specify the settings for blockage generation and for loading or saving an options file.
  5. Use Cover Blockage Layers to specify layers and choose Detailed Blockage Layers to specify a layer-geometry specification.
  6. Select Load/Save Options File to load or save the options file. You can use the following methods to load and save an options file:
    • Save the abstraction rules specified in the Abstract Pcells form to an options file.
      • Click Save to save the abstraction rules to an options file.
      • Specify the name of the options file to which you want to save the abstraction rules in the Options File Name field. You can also load an existing options file, edit it, and save the changes.
    • Load the various rules contained in an options file to the Abstract Pcells form. To load the abstraction rules contained in an options file, select the required options file and click Load.
