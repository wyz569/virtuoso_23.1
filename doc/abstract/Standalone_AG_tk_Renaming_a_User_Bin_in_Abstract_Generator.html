Renaming a User Bin in Abstract Generator

Bins are like buckets for placing cells. Each cell in a library is always contained in one bin. Each bin is associated with certain options, which allows the cells in a library to be distributed into mutually exclusive sets for processing.

To change the name of the selected user bin:

  1. Choose Bins – Rename.
  2. In Enter new bin name, type the new name for the selected user bin and
  3. Click OK to accept the change.
You cannot change the names of system bins.
