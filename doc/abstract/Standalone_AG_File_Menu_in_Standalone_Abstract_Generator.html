File Menu in Standalone Abstract Generator

You can use the commands in the File menu to specify all setup information, ranging from defining technology information and specifying a library through importing physical and logical data for the cells and blocks to be processed, for Abstract Generator to process a library. The commands for specifying this information are also available on the toolbar.

You can open and create a library, attach technology information to a library in Abstract Generator. The commands for importing GDSII information, OASIS data, LEF and DEF are also available on the File menu. Other commands in the File menu let you export LEF descriptions of abstracts, generate a statistics report for the current library, and save the option settings of the current session. You can also replay a previous session.
