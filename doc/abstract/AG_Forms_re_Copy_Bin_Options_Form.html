Copy Bin Options Form

Use the Copy Bin Options form to copy option settings from another bin to the selected bin. This is useful when creating a new user bin; you can selectively copy bin specific options only while copying options from one bin to another.

Field Description

From

Lets you select the bin whose options you want to copy. The available options are Core, IO, Corner, and Block. For example, the settings for the Core bin could be copied to UserBin.

To

Lets you enter the current bin.
