Specify SKILL Files Form

Use the Specify SKILL Files form to customize the Abstract Generator flow for specific tasks that cannot be performed using the options in the Abstract Generator form.

Field Description

Add

Displays the Choose Skill Files form, which lets you select the required skill hook files.

Remove

Removes the selected SKILL hook files.

OK

Saves the selection and closes the Specify SKILL Files form.
