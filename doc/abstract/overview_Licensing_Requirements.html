Abstract Generator Licensing Requirements

Standalone Abstract Generator

The following figure describes the licensing requirements for standalone Abstract Generator:

Integrated Abstract Generator

Required licenses:

  • DFII framework '111' license
  • Virtuoso Advanced Node Framework '95011' license
  • One of the following:
    • Virtuoso Layout Suite XL license + 2 GXL tokens
    • Virtuoso Layout Suite GXL (4 tokens) + 2 GXL tokens
    • Virtuoso Layout Suite EXL

The following image captures the licensing behavior of the integrated Abstract Generator.

If Abstract Generator has checked out one of the above mentioned license, then it does not check out any other additional licenses.

Additionally, when a library is opened, if the technology file has data reflecting geometries of 20nm, 16nm, or 14nm, the Virtuoso_Adv_Node_Opt_Lay_Std license (95512) is checked out. If the technology file has data reflecting geometries of 10nm or smaller, the Virtuoso_Adv_Node_Opt_Layout license (95511) is checked out.
