Viewing and Editing Cell Properties

You can use the Cell Properties form to view and change the properties on cells.

To change a property of a cell:

  1. Select Cells – Cell Properties after selecting one or more cells in the Cell pane.
    The selected cells are listed in the Cell Properties form. Also listed is the bin in which the cell is stored and the current settings for the prCellClass and symmetry properties. You can select more than one cell by using the usual Shift and Ctrl key bindings.
    Abstract Generator is able to set properties only on valid views that it has created (Pins, Extract and Abstract). If a cell contains none of these views or if the views are invalid, the entries in the table are grayed out and you cannot change the properties for that cell.
  2. Select one or more cells in the list and use the Change property cyclic field to select the property you want to change.
    This changes the Allowed Values in the second cyclic field.
  3. Select the new value for the selected cells.
  4. Click Apply to update the form. This will not update the database.
  5. Click OK to accept all your changes and close the form.
