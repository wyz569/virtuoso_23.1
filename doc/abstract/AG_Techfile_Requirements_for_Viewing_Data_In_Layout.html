Requirements for Viewing Data In Layout

The section describes the technology file requirements for viewing your data using Virtuoso tools.

Display packets are defined in the display resource file and control how layers appear on your monitor and plotting devices. You assign a display packet to a particular layer-purpose pair and specify other layer display attributes in the techDisplays subsection of the layerDefinitions section.
