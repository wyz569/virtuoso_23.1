Import DEF Form

Use the Import DEF form to import a net list in DEF format and update the design with the information.

Field Description

DEF Filenames

Specifies the DEF files to import. You can type in the name manually or Browse for existing files with the .def extension in the current directory.

Default Bin

Lets you select the bin to which DEF data is to be imported. Cells are placed in the Default Bin only if they confirm to the following criteria:

  • There is a valid layout view.
  • The cell is not a via.
  • The cell is not a site.

If any of this criteria is not met, the cells will be placed in the Ignore bin.
