Select Cells Form

Use the Select Cells form to select one or more cells using a variety of selection criteria.

Field Description

Name

Lets you enter a regular expression that is included in the cell names that you selected. All matching cells are selected, irrespective of the View set.

View

Lets you select the view you are interested in. Only cells that have the specified View present and match any subsequent Height, Property or Status criteria are selected.

Height

Lets you specify minimum or maximum cell heights, or a range within which the cell height must fall.

Property

Lets you enter a system or user-defined cell property and a value for the cells you want to select.

Select By Error Messages

Lets you select one or more of the error messages listed in the table to select cells in a library. Only cells that have had a specific, selected error message applied to them will be filtered through in the selection procedure.

Name

Lets you enter a regular expression that is included in the cell names you want to select. All matching cells are selected, irrespective of the View set.
