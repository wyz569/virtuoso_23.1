New Library Window

Use the New Library window to create a new library in the specified directory. You can create a library at any time during the abstract generation process.

Field Description

Library Path

Specifies a valid directory path to where the new library should be created. The default value is the current directory, that is, “.”.

Browse

Displays the Browse New Library Directory form that can be used to locate the required directory location for the new library. If no path is specified, a warning message is displayed requesting that you enter a valid directory path.

Library Name

Specifies a valid library name. The library name must not contain any spaces. You also cannot specify the name of an existing library that is already specified in the cds.lib file of the current directory.
