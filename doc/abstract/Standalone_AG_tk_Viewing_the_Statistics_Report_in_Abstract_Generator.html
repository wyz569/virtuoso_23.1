Viewing the Statistics Report in Abstract Generator

At any point during the Abstract Generator run, use the statistics report to see a report window that displays the abstract generation status and statistics until that point.

  1. Choose File – Statistics Report.
    A separate statistics report is displayed for each selected cell.
  2. The < and > buttons are useful when multiple cells are selected for abstract generation. Use these buttons to navigate between reports for the various cells. A new statistics report is generated for each Abstract Generator run.
  3. Click Save As. The Save Statistics window is displayed.
  4. Specify a different Cell Stats Filename. The default filename is cellSummary.report.
  5. To select an existing file, click Browse and select the required file.
    If a file with the specified name already exists, then a message indicating this is displayed. You can choose to either replace the existing file or save the report with a different filename.
  6. Click OK.
    The statistics report is stored in the text format.
