Export Abstracts

When you finish generating abstract cellviews, you can export the data in the form of LEF abstracts for use in place-and-route tools.

The File – Export – LEF command provides the functionality to translate cell abstracts into LEF, which can be used as input to place-and-route tools.

The File – Export – Options command lets you save bin options settings for reuse in a later session. This saves all the current option settings to an.abstract.options file in your current working directory.

Abstract Generator writes the following system files in the library directory:

  • .abstract.options: Stores Abstract Generator option settings for the library. This file is available only in the SKILL format.
  • .abstract.cellstatus: Stores the Abstract Generator status for all cells in the library. For example, a cellview might have the status exist, exists with errors, exists with warnings, and so on.
