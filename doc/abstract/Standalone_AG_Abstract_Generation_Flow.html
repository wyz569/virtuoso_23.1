Abstract Generation Flow

This topic outlines the basic steps required to start generating abstracts for cells or libraries. The following image depicts a flow for generating abstracts in a design.

Before beginning with the abstract generation process, you need to perform the following operations:

  1. Open a library.
  2. Attach technology information.
  3. Import layout information.
  4. Import logical information.

Not all of these procedures are compulsory; see individual procedure for details.
