Troubleshoot Library Issues

You can face following issues while attaching a technology information to a library in Abstract Generator.

ABS-10517

The name library has been interrupted while loading. value cells out of value have been successfully loaded.

To resolve this issue:

  • Load the cell library again without interrupting Abstract Generator.

ABS-10526

Failed to attach the '%s' technology library to the '%s' name library. Ensure the following:

  1. Use the absAttachTechLib command after setting a library. You can do this only in nogui or replay modes.
  2. You have permission to write in the design library, specifically the prop.xx file, which contains property information regarding the attached technology library.
  3. The design library exists and that it is not locked.
  4. The technology library you are trying to attach exists.
