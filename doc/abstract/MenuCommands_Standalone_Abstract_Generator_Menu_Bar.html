Standalone Abstract Generator Menu Bar

This topic describes the menu commands and the associated forms in the standalone Abstract Generator. Abstract Generator does not check the validity or correctness of the entered text, such as terminal name or layer name, against any syntax rules.

Abstract Generator contains File, Bins, Cells and Flow menu items.

The following table describes the types of options available in the menu bar.

Menu Types of Options

File

Use the commands in the File menu to specify all setup information, ranging from defining technology information and specifying a library through importing physical and logical data for the cells and blocks to be processed, for Abstract Generator to process a library. The commands for specifying this information are also available on the toolbar.

Other commands in the File menu let you export LEF descriptions of abstracts, generate a statistics report for the current library, and save the option settings of the current session. You can also replay a previous session.

Bins

Use the commands in the Bins menu to create, rename, and delete user bins, and to access the Bin Options form.

Cells

Use the commands in the Cells menu to select and sort the cells displayed in the cell table, move cells between different bins, view cell status reports, and set cell and terminal properties. The Edit submenu opens the Layout Editor, which you can use to inspect or modify any cellview.

Flow

Use the commands in the Flow menu to provide access to the main abstract generation flow commands: Pins, Extract, and Abstract. You can also access these commands by using the last four buttons on the toolbar.

The flow options are available only when technology data has been successfully validated by the Abstract Generator technology checker and you have selected one or more library cells to be processed.
