Standalone Abstract Generator

The standalone Abstract Generator interface is designed to be sufficiently intuitive to allow first-time users to make good progress and at the same time provide the flexibility required to satisfy the goals of expert users. The interface provides comprehensive status information and supports the recommended use model by allowing backward steps in the flow and providing helpful feedback at all stages.

You can launch Abstract Generator and use the flow defined by the tool to generate abstracts from layout data. You can use the commands, options and graphical elements to describe the features that you encounter while using the interface. Some abstract generation functions can adversely affect the performance of the tool and should be run only when necessary. Use the guidelines to optimize the performance of Abstract Generator.
