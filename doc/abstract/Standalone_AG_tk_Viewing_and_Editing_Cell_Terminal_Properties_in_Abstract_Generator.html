Viewing and Editing Cell Terminal Properties in Abstract Generator

The Terminal Properties form lets you edit terminal properties for the cells in your library. Abstract Generator assigns the properties to the terminals in the database. If the logical information already contains these properties on terminals, Abstract Generator displays them in the Terminal Properties form and propagates them through various Abstract Generator views. This form works in the same way as the Cell Properties Form.

Abstract Generator is able to set properties only on valid views that it has created (Pins, Extract and Abstract). If a cell contains none of these views or if the views are invalid, the entries in the table are grayed out and you cannot change the properties for the terminals in question.

To change a terminal property of a cell, perform the following steps:

  1. Choose Cells – Terminal Properties after selecting one or more cells in the Cell pane.
    The Terminal Properties form is displayed. The table contains an entry for each terminal (Terminal column) defined for each of the selected cells (Cell column) selected in the Cell pane.
  2. Select one or more terminals from the list and use Change property to select the property you want to change.This changes the Allowed Values in the second cyclic field.
  3. Select the new value for the selected cells.
  4. Click Apply to make the change. This updates the form but not the database.
  5. Click OK to accept all your changes and close the form.
Terminal property settings are saved in the view of the current step. Therefore, when a step is run again, the terminal properties will be restored to the original values set prior to any user amendments.
