Creating a User Bin in Abstract Generator

You can create user bins in addition to the existing system bins. User bins differ from system bins only in that you can delete and rename user bins, while system bins are permanent and cannot be renamed or deleted. The commands in the Bins menu apply only to user bins.

To create a user bin:

  1. Load a library and choose Bins – Add from the menu bar.
  2. Enter a name and set a type for the new bin by selecting a choice from the Choose bin type drop-down box.
    The bin type is important because it determines the default values for certain options.
  3. Click OK.

The new bin is created at the bottom of the list in the Bin pane.
