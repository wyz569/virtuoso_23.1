Axis Editor Form

To open the Axis Editor form, click the Axis column in the Symmetry tab. This form lets you edit the axis direction.

Field Description

Name

The name for the given axis constraint (multiple symmetries can use the same axis constraint).

Direction

The orientation of the axis of symmetry for the symmetry members. The valid values are vertical and horizontal.

Location

The distance from the design centerline (in x direction for vertical axis direction or y direction with horizontal axis direction) from which the symmetry plane extends.
