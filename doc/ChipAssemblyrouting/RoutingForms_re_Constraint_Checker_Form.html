Constraint Checker Form

To open the Constraint Checker form, click the Check button on the Routing Constraint Manager toolbar. This form lets you specify the options for constraint checking.

Field Description

Hier Depth

Specifies a value to adjust the depth of checks. The checks are performed on the less than or equal to specified depth value.

Selected Nets Only

Performs the constraint checks only on the selected nets.

Per Layer

Checks for the matched length constraint.

Preserver Marker

Checks that the existing markers are not removed.

Report Hierarchy Violations

Checks the constraint down in the hierarchy and reports any existing violations.

Constraint Checks

Lets you select the constraints for which the checks are to be performed.

Checks

Enables all constraint checks when selected. The checks can be performed for the following constraints.

  • Process Rule Overrides
  • Symmetry
  • Match
  • Shield

You can also perform checks on the selected constraints.
