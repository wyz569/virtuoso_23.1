Specify Routed Cellview

The Specify Routed Cellview lets you specify a cellview that you want to route.

Field Description

Library

Lets you select a library.

Cell

Lets you select a cell to be routed. You can also specify a non existing name.

View

Lets you select a view for the selected library and cell. You can also specify a non existing name.
