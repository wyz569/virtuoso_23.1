dbDetachPRBoundaryFromRowRegion

dbDetachPRBoundaryFromRowRegion(
d_PRBoundaryId
) 
=> t / nil

Description

Detaches the PRBoundary from any associated rowRegion.

Arguments

d_PRBoundaryId

Datbase ID of PRBoundary.

Value Returned

t

PRBoundary is detached from the rowRegion.

nil

Returned in case of failure.

Example

Creates a row region and PR boundary and attaches them. dbDetachPRBoundaryFromRowRegion is then used to detach them.

         rowRegion = dbCreateRowRegion(cv "rowRegion" "newRowRegionSpec" point)
         prb = dbCreatePRBoundary(cv list(list(0 0) list(100 0) list(100 100) list(0 100)))
         ;; Attach RowRegion to PRBoundary and detach PRBoundary from RowRegion.
         dbAttachRowRegionToPRBoundary(rowRegion)
         dbDetachPRBoundaryFromRowRegion(prb)
