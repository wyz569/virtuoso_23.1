pkxAddFuncDefPin

pkxAddFuncDefPin(
pkx_compDefPinId
pkx_funcDefPinId
) 
=> t / nil

Description

(Virtuoso MultiTech Framework) Adds the specified funcDefPin object to the specified compDefPin object.

Arguments

pkx_compDefPinId

ID of a compDefPin object.

pkx_funcDefPinId

ID of a funcDefPin object.

Value Returned

t

The specified funcDefPin is added to compDefPin.

nil

The operation failed.

Example

Adds funcDefPin to compDefPin.

pkxAddFuncDefPin(compDefPin funcDefPin)
