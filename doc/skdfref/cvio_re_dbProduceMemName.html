dbProduceMemName

dbProduceMemName( 
t_name 
) 
=> list( <memName> ... <memName> ) / nil

Description

Returns a list of member names for the given name.

Arguments

t_name

Name whose members needs to be generated.

Values Returned

list

Returns list of the member names.

<memName>

Member names.

nil

There are errors.

Examples

Returns a list of member names for "7".

dbProduceMemName("7")
=>("7")

Returns a list of member names for "(X)".

dbProduceMemName("(X)")
=>("X")

Returns a list of member names for "(7)".

dbProduceMemName("(7)")
 ("7")

Returns a list of member names for "()".

dbProduceMemName("()")
=>nil

Returns a list of member names for "x(a)".

dbProduceMemName("x(a)")
=>("x#28a#29")

Returns a list of member names for "x(3)".

dbProduceMemName("x(3)")
=>("x(3)")

Returns a list of member names for "(a)x".

dbProduceMemName("(a)x")
=>nil

Returns a list of member names for "(2)x".

dbProduceMemName("(2)x")
=>nil

Returns a list of member names for "<*3>(0)".

dbProduceMemName("<*3>(0)")
=>("0" "0" "0")

Returns a list of member names for "<*3>(A)".

dbProduceMemName("<*3>(A)")
 ("A" "A" "A")
