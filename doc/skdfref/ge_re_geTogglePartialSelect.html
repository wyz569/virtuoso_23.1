geTogglePartialSelect

geTogglePartialSelect( 
)
=> t / nil

Description

Toggles partial selection off and on.

This is the equivalent of pressing the F4 key.

Arguments

None

Value Returned

t

partialSelect is on.

nil

partialSelect is off.
