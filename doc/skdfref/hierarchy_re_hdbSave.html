hdbSave

hdbSave(
h_configId
)
=> t / nil

Description

Saves a configuration opened with hdbOpen.

Arguments

h_configId

A configuration ID returned from a call to hdbOpen.

Value Returned

t

The configuration is saved.

nil

The save operation failed.

Example

hdbSave( hdbConfig )
