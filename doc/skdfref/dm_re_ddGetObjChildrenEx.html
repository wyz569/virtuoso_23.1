ddGetObjChildrenEx

ddGetObjChildrenEx(
b_objId 
x_pathType
) 
=> l_children / nil

Description

Returns the children of b_objId. This function is similar to ddGetObjChildren except that you can specify whether to look for the object in the temporary directory or the master directory, or both, with the x_pathType argument.

Arguments

b_objId

Object ID returned by ddCreateLib or ddGetObj.

x_pathType

One of the following integer values:

    • 0 Looks for the object only in the master directory and, if found, returns the children from the master directory only
    • 1 Looks for the object only in the temporary directory and, if found, returns the children from the temporary directory only
    • 2 Returns all the children in both the master and temporary directories

Value Returned

l_children

The IDs of the children of b_objId.

nil

b_objId does not exist or does not have any children.
