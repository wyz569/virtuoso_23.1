Utility Functions

You can use the utility functions to perform tasks such as:

  • Open a design in a new window, verify the database ID of a window, and create a copy of a cellview.
  • Display the properties of each object in the selected set.
  • Determine whether a path or polygon is self-intersecting.
  • Search a point list in a cellview for collinear points and delete the intermediate collinear points.
  • Create or update a DFII display packet.
