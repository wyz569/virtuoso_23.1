Vector Instance Definition Attributes

Attribute Modify Description
Include Generic Object Attributes.

baseName

no

string

Returns base name for the vectorInstDef.

minIndex

no

integer

Returns smallest bit number in use for the vectorInstDef.

maxIndex

no

integer

Returns largest bit number in use for the vectorInstDef.

bitOrder

no

integer

Returns or sets bit order for the vectorInstDef. If the vectorInstDef is implicit, “none” will be returned. bitOrder can be set if the bus definition is explicit.

Valid values for modification:
String value.

numBits

no

integer

Returns the number of bits covered for this vectorInstDef.

numBitsUsed

no

integer

Returns the number of vectorInstBits used for this vectorInstDef.
