geDiscardEdits

geDiscardEdits( 
[ w_windowId ] 
) 
=> t / nil

Description

Discards all edits made to a design.

Arguments

w_windowId

Database ID of the window for which edits are to be discarded. If not specified, the current window is used.

Value Returned

t

The edits are discarded.

nil

The edits are not discarded.

Examples

Discards the edits in win3.

geDiscardEdits( win3 )
