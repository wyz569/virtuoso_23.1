dbGetTrunkLongOffset1

dbGetTrunkLongOffset1(
d_trunkId 
)
=> f_longitudinalOffset1 / nil

Description

Returns the longitudinal offset1 value of the specified trunk pattern ID.

Arguments

d_trunkId

The trunk pattern ID whose longitudinal offset1 value must be returned.

Value Returned

f_longitudinalOffset1

  

The longitudinal offset1 value of the specified trunk pattern ID.

nil

The trunk pattern ID is incorrect.

Examples

dbGetTrunkLongOffset1(trunk1)
