dbSetInPlaceCoverObstructionPinRemodeling

dbSetInPlaceCoverObstructionPinRemodeling(
d_InstHeaderId
t_remodelingName
) 
=> t / nil

Description

Sets value of the InPlaceCoverObstruction PinRemodeling attribute on the specified instance header.

Arguments

d_InstHeaderId

ID of an instance header.

t_remodelingName

Valid PinRemodeling attribute value.

Values Returned

t

The specified InPlaceCoverObstruction PinRemodeling attribute Value is set on the specified instance.

nil

The operation failed.

Examples

Sets value of the InPlaceCoverObstruction PinRemodeling attribute "perimeterOnly" on the instance header instHeaderId.

inst = dbCreateInst(cv master "inst" list(3 4) "R0" 1)

instHeaderId = inst~>instHeader

dbCreateInPlaceCoverObstruction(instHeaderId list("closed" 1 4))

dbSetInPlaceCoverObstructionPinRemodeling(instHeaderId "perimeterOnly")

equal(dbGetInPlaceCoverObstructionPinRemodeling(instHeaderId) "perimeterOnly")
