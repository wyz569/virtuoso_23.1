Layer Boolean Functions

Layer Boolean SKILL functions let you create certain geometries using Boolean operators, such as AND and perform relational selection operations such as ENCLOSE on existing geometries.

These SKILL functions take input from one or two lists of database objects of shapes or polygons while the output of the function is created as database objects on a specified output layer. The object IDs of the created database objects are returned.

This topic provides a list of Cadence® SKILL functions associated with the layer Boolean functions.

Only the functions listed here are supported for public use. All other functions, regardless of their name or prefix, and undocumented aspects of the functions described below, are private and are subject to change at anytime.
