Photonics Symbol Generator Functions

The optional SKILL generators speed up the generation of consistent symbols for photonics PDK. They can be used to generate a regular symbol cellview with a single call instead of drawing the symbol. They can also be wrapped into a SKILL Pcell shell, which allows support for magnification.

These generators check whether the Virtuoso Photonics Option is enabled. For more information on starting Virtuoso with photonics enabled, refer to

Ports for these symbols are typed "optical" to enable optical signal handling in Virtuoso Schematic Editor.

All the functions allow the PDK developer to specify the name of the optical (and electrical, if relevant) ports and direction.
