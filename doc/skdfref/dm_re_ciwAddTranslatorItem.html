ciwAddTranslatorItem

ciwAddTranslatorItem(
s_tag
t_itemText
t_callback
)
=> t / nil

Description

Adds an entry for the Import or Export commands.

Arguments

s_tag

A SKILL symbol.

Valid Values: ‘Import, ‘Export

t_itemText

The Item text to be used as a display prompt.

t_callback

Called to execute the given translator.

Value Returned

t

The entry was added.

nil

The entry was not added.
