Dot Attributes

A dot consists of a point. It can be used for marking locations, such as pin location, feed through, and so forth.

Attribute Modify Type Description
include Generic Shape Attributes.

height

yes

float

Height of the dot.

width

yes

float

Width of the dot.

xy

yes

point

Point describing dot's location (in user units).
