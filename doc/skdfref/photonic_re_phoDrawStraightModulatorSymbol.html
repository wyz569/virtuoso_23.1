phoDrawStraightModulatorSymbol

phoDrawStraightModulatorSymbol(
d_cellViewId
n_magnification
?startPortName s_PortName
?startPortDirection s_direction
?endPortName s_PortName
?endPortDirection s_direction
?cathodeName s_name
?cathodeDirection s_direction
?anodeName s_name
?anodeDirection s_direction
)
=> t / nil

Description

(Virtuoso Photonics Option) Draws a straight modulator symbol.

Arguments

d_cellViewId

Database ID of a cellview.

n_magnification

Database magnification value to set on the instance.

Default: 1.0

?startPortName s_PortName

Name of the starting port. The default is "opt0".

?startPortDirection s_direction

Direction of the starting port. The default is "inputOutput".

?endPortName s_PortName

Name of the ending port. The default is "opt1".

?endPortDirection s_direction

Direction of the ending port. The default is "inputOutput".

?cathodeName s_Name

Name of the cathode. The default is "CA".

?cathodeDirection s_direction

Direction of the cathode. The default is "inputOutput".

?anodeName s_Name

Name of the anode. The default is "AN".

?anodeDirection s_direction

Direction of the anode. The default is "inputOutput".

Return Value

t

Symbol for the straight modulator is generated.

nil

Symbol is not generated.

Example

cv = geGetEditCellView()
phoDrawStraightModulatorSymbol(cv 1.0)
