ddsHiCloseData

ddsHiCloseData(
)
=> t / nil

Description

Displays the Close and Purge Data form that prompts you to select one or more objects to close and purge form virtual memory.

The Close and Purge Data form can display data that have lost connection to their physical files due to one of the following reasons:

  • A network failure, due to which data on the shared file servers becomes inaccessible
  • Deletion of local data
  • Renaming of local data

The form lets you purge such data from the virtual memory. It displays relevant warnings in CIW.

Arguments

None

Value Returned

t

The function succeeded.

nil

The function did not succeed.
