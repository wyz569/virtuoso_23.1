hdbIsOpenConfig

hdbIsOpenConfig(
h_configID
)
=> t / nil

Description

Checks if the configuration ID you specify is valid. The configuration ID is valid if the configuration is still open.

Arguments

h_configID

A configuration ID, returned by hdbOpen.

Value Returned

t

The configuration ID is valid.

nil

The configuration ID is not valid.
