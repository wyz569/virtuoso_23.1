geRefreshCellView

geRefreshCellView(
d_cvId 
l_Bbox
)
=> t / nil

Description

Redraws a cellview when necessary. If a bounding box is specified, only the specified area is redrawn.

Arguments

d_cvId

ID of the cellview to be redrawn.

l_Bbox

List of points that specify an area to be redrawn.

Value Returned

t

Cellview or area is successfully refreshed.

nil

Cellview or area is not refreshed.
