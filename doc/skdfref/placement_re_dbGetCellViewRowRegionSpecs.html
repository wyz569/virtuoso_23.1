dbGetCellViewRowRegionSpecs

dbGetCellViewRowRegionSpecs(
d_cellviewId
)
=> l_rowRegionSpecs / nil

Description

Returns a list of rowRegionSpecs in the specified cellview.

Arguments

d_cellviewId

Database ID of a cellview.

Value Returned

l_rowRegionSpecs

List of database IDs of the rowRegionSpecs in the given cellview.

nil

Returned in case of failure.

Example

Returns a list of RowRegionSpecs in the cellview cv.

dbGetCellViewRowRegionSpecs(cv)
