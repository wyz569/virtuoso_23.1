dbGetRowRailDef

dbGetRowRailDef(
d_rowId
)
=> l_dbRailDefs / nil

Description

Returns a list of dbRailDefs referenced in the specified row.

Arguments

d_rowId

Database ID of the row.

Value Returned

l_dbRailDefs

A list of dbRailDefs is returned.

nil

The row ID is invalid

Examples

Returns a list of dbRailDefs referenced in the given row.

row = car(geGetEditCellView()~>rows)
myRailDefs = dbGetRowRailDef(row)
