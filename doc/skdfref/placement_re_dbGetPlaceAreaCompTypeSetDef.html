dbGetPlaceAreaCompTypeSetDef

dbGetPlaceAreaCompTypeSetDef(
d_customPlacementArea 
)
=> l_dbCompTypeSetDefs / nil

Description

Returns a list of dbCompTypeSetDefs referenced in the specified placeArea.

Arguments

d_customPlacementArea

  

Database ID of the custom placeArea.

Value Returned

l_dbCompTypeSetDefs

  

A list of dbCompTypeSetDefs is returned.

nil

The custom placeArea ID is invalid or no dbCompTypeSetDefs exist.

Examples

Returns a list of database IDs of the background definitions in the placement area placeAreaId.

placeArea = car(dbGetCellViewPlaceArea(geGetEditCellView()))
myCompTypeSetDefs = dbGetPlaceAreaCompTypeSetDef(placeArea)
