dbConvertEllipseToPolygon

dbConvertEllipseToPolygon( 
d_ellipse 
x_nSides 
) 
=> d_polygon / nil

Description

Converts an ellipse to a polygon.

Arguments

d_ellipse

The ellipse to be converted.

x_nSides

Number of sides defining the polygon.

Value Returned

d_polygon

The dbObject of the polygon if the conversion is successful.

nil

The ellipse is not converted.

Examples

dbConvertEllipseToPolygon(ellipseId 7)
