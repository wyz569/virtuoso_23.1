ccOREGetContexts

ccOREGetContexts(
)
=> l_contextNames

Description

Retrieves a list of Optical Rule Engine (ORE) context names.

Arguments

None

Value Returned

l_contextNames

(t_contextName...)

A list of ORE context names.

Example

Define variable PI_2 and save the resulting ORE context under the name ore_cxt.

ccORE(?rule '((define PI_2 PI/2)) ?saveContext "ore_cxt")
=> t

Get the list of the names of the ORE contexts.

ccOREGetContexts()=> ("ore_cxt")
