geGetEditCellViewWindow

geGetEditCellViewWindow( 
d_cellview
)
=> w_windowId / nil

Description

Returns the ID of the window in which the specified cellview is the edit cellview.

Arguments

d_cellview

Database ID of the cellview.

Value Returned

w_windowId

Database ID of the window in which the specified cellview is the edit cellview.

nil

An edit cellview window was not found or an error occurred.
