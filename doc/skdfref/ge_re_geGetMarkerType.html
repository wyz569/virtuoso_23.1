geGetMarkerType

geGetMarkerType( 
d_figId 
)
=> t_type / nil

Description

Displays the type of a marker.

Arguments

d_figId

Database ID of the marker.

Value Returned

t_type

The marker type, either warning or error.

nil

The marker is not found.

Examples

Returns the marker type of marker2.

geGetMarkerType( marker2 )
