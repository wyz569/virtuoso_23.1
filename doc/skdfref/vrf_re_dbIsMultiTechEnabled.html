dbIsMultiTechEnabled

dbIsMultiTechEnabled(
)
=> t / nil

Description

(Virtuoso MultiTech Framework) Confirms whether the Virtuoso Multi Tech Framework functionality is enabled in the current session.

This function will be deprecated in a future release.

Arguments

None.

Value Returned

t

Returns t when all of the following are true:

  • Virtuoso Multi Tech Framework is available
  • Virtuoso Multi Tech Framework license has been checked out
  • An advanced node and methodologists release, such as ICADVM20.1 is being used

nil

Virtuoso Multi Technology Solution framework is not enabled or you are not using an advanced node and methodologies release such as ICADVM20.1.

Example 

dbIsMultiTechEnabled()
=> t
