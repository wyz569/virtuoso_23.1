Row Attributes

Attribute Modify Type Description
include Generic Figure Attributes.

yes

string

Name of the specified row.

no

objId

Database ID of the rowHeader associated with the specified row.

yes

objId

Database ID of the siteDef associated with the specified row.

yes

string

yes

Number of sites for the specified row.

yes

Orientation of the row.
Valid values for modification:
R0, R90, R180, R270, MY, MYR90, MX, MXR90.
