dbGetTerminalAntennaRuleOnLayer

dbGetTerminalAntennaRuleOnLayer( 
d_termId 
t_modelName
t_dataName
tx_layer
)
=> g_value / nil

Description

Returns an antenna rule of the terminal according to the model name, data name, and layer.

Arguments

d_termId

The database ID of the terminal.

t_modelName

Specifies the model name.

Valid values: "defaultAntennaModel", "secondAntennaModel", "thirdAntennaModel", "fourthAntennaModel","antennaModel5" to "antennaModel32"

t_dataName

Specifies the data name. Valid values:

"ANTENNAGATEAREA"
"ANTENNADIFFAREA"
"ANTENNAPARTIALMETALAREA"
"ANTENNAPARTIALMETALSIDEAREA"
"ANTENNAPARTIALCUTAREA"
"ANTENNAMAXAREACAR"
"ANTENNAMAXSIDEAREACAR"
"ANTENNAMAXCUTCAR"

tx_layer

The layer name.

Value Returned

g_value

The value of the antenna rule is returned.

nil

An error occurred during execution.

Example

dbGetTerminalAntennaRuleOnLayer(termA "defaultAntennaModel" "ANTENNAGATEAREA" "Metal1")
