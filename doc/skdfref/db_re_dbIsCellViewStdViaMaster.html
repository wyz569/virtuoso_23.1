dbIsCellViewStdViaMaster

dbIsCellViewStdViaMaster( 
d_cellViewId
) 
=> t / nil

Description

Returns a boolean value indicating whether the specified cellview is a standard via master.

Arguments

d_cellViewId

The database ID of the cellview.

Value Returned

t

The specified cellview is a valid standard via master.

nil

The specified cellview is a regular cellview or the cellview ID is invalid.
