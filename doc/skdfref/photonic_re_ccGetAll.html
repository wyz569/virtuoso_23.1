ccGetAll

ccGetAll(
d_cellViewId
)
=> l_shapes / nil

Description

(Virtuoso Photonics Option) Returns all persistent ccShapes in the specified cellview.

Arguments

d_cellViewId

Database ID of a cellview.

Value Returned

l_shapes

List of ccShape IDs.

nil

An error occurred.

Example

ccGetAll(cv)
=> (cc@0x269c83f0)
