Multi-Patterning Technology Database Access Functions

Multi-patterning technology uses two or more mask processes to manufacture a design layer. Colors are used to represent the individual masks. This topic describes the SKILL functions that support photo mask coloring of objects and coloring methods for MPT.

For more information about multi-patterning lithography and how to use Virtuoso® Multi-Patterning Technology (MPT) for designs that require two or more masks for a drawn layer, see Virtuoso® Multi-Patterning Technology User Guide in the ICADVM20.1 release.
