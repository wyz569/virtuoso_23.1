ccIsLinearCurve

ccIsLinearCurve(
o_curve
)
=> t / nil

Description

(Virtuoso Photonics Option) Checks if the given ccCurve is linear.

Arguments

o_curve

ID of a ccCurve object.

Value Returned

t

The specified curve is linear.

nil

The specified curve is not linear.

Example

ccIsLinearCurve(ccCreateCurve("t" "1" 0:1))
=> t
ccIsLinearCurve(ccCreateCurve("cos(t)" "sin(t)" -3.14:3.14))
=> nil
