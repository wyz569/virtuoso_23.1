dbStartGenNetToInstTerm

dbStartGenNetToInstTerm(
d_netId
)
=> I_genId

Description

Returns a dbGenerator object that can be used to iterate over all instance terminals of the specified net.

Arguments

d_netId

Database ID of a net.

Value Returned

I_genId

ID of a dbGenerator object.

Example 

gen1 = dbStartGenNetToInstTerm(net1)
while(instTermId = gen1~>next
;Use instTermId in the code
)
; Close dbGenerator object reference and remove it from virtual memory
dbStopGen(gen1)
