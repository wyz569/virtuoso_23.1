phoDrawWayPointConnectorSymbol

phoDrawWayPointConnectorSymbol(
d_cellViewId
n_magnification
?startPortName s_PortName
?startPortDirection s_direction
?endPortName s_PortName
?endPortDirection s_direction
)
=> t / nil

Description

(Virtuoso Photonics Option) Draws a waypoint connector symbol.

Arguments

d_cellViewId

Database ID of a cellview.

n_magnification

Database magnification value to set on the instance.

Default: 1.0

?startPortName s_PortName

Name of the start port. The default is "opt0".

?startPortDirection s_direction

Direction of the start port. The default is "inputOutput".

?endPortName s_PortName

Name of the end port. The default is "opt1".

?endPortDirection s_direction

Direction of the end port. The default is "inputOutput".

Return Value

t

Symbol for the waypoint connector is generated.

nil

Symbol is not generated.

Example

cv = geGetEditCellView()
phoDrawWayPointConnectorSymbol(cv 1.0)
