dbGetPlaceBoundMinHeight

dbGetPlaceBoundMinHeight( 
d_placeBound
)
=> x_minHeight / nil

Description

(Virtuoso MultiTech Framework Only) Returns the minimum thickness of a placeBound.

Arguments

d_placeBound

  

The placeBound pointer.

Value Returned

x_minHeight

The minimum thickness of a die.

nil

The minimum thickness could not be obtained.

Example 

placeBound1 = (dbCreatePlaceBound cv "pb1" list(0:0 100:100) 0 100)
\i (dbGetPlaceBoundMinHeight placeBound1)
\t 75
