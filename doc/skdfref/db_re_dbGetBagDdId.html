dbGetBagDdId

dbGetBagDdId( 
g_bagId 
) 
=> d_ddId / nil

Description

Returns the ddId associated with the specified bagId.

Arguments

g_bagId

dbBagId object.

Value Returned

d_ddId

The property bag’s dd ID.

nil

The given g_bagId is invalid.

Examples

dbGetBagDdId( bagId )
