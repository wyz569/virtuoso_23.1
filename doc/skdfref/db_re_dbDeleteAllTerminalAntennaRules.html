dbDeleteAllTerminalAntennaRules

dbDeleteAllTerminalAntennaRules( 
d_termId 
t_modelName 
)
=> t / nil

Description

Deletes all antenna rules from the specified terminal.

Arguments

d_termId

The database ID of the terminal.

t_modelName

The name of the antenna model.

Valid values: "defaultAntennaModel", "secondAntennaModel", "thirdAntennaModel", "fourthAntennaModel","antennaModel5" to "antennaModel32"

Value Returned

t

The figure was added successfully.

nil

An error occurred during execution.

Example

dbDeleteAllTerminalAntennaRules(termA "defaultAntennaModel")
dbDeleteAllTerminalAntennaRules(termA "secondAntennaModel")
