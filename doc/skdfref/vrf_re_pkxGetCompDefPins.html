pkxGetCompDefPins

pkxGetCompDefPins(
d_bagId
) 
=> list(pkx_compDefPinId) | nil

Description

(Virtuoso MultiTech Framework) Returns a list of compDefPin object IDs in the specified database.

Arguments

d_bagId

ID of the database containing the compDefPin objects.

Value Returned

list(pkx_compDefPinId)

List of compDefPin objects.

nil

Cannot find any compDefPin objects or the operation failed.

Example

Returns a list of compDefPin objects in the database bag. 

pins = pkxGetCompDefPins(bag)
