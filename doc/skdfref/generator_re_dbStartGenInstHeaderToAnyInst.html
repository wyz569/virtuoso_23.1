dbStartGenInstHeaderToAnyInst

dbStartGenInstHeaderToAnyInst(
d_instHeaderId
)
=> I_genId

Description

Returns a dbGenerator object that can be used to iterate over all instances of the specified instance header.

Arguments

d_instHeaderId

Database ID of an instance header.

Value Returned

I_genId

ID of a dbGenerator object.

Example 

gen1 = dbStartGenInstHeaderToAnyInst(instHeader1)
while(instId = gen1~>next
;Use instId in the code
)
; Close dbGenerator object reference and remove it from virtual memory
dbStopGen(gen1)
