ddCatClose

ddCatClose( 
b_ddId 
) 
=> t / nil

Description

Closes the category and purges its information from virtual memory. Any edits since the last ddCatSave are lost.

Arguments

b_ddId

ID of a category.

Value Returned

t

The function was successful.

nil

The function failed.
