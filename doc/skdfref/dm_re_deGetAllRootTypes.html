deGetAllRootTypes 

deGetAllRootTypes( 
)
=> l_rootTypes / nil

Description

Returns a list of all currently registered root view types.

Arguments

None

Value Returned

l_rootTypes

List of all registered root view types.

nil

The function failed.

Examples

deGetAllRootTypes()

Returns ("graphic" "maskLayout" "schematic").

"TimingAnalysis" is not in the list because it is a subtype of "schematic".
