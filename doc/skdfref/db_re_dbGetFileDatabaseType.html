dbGetFileDatabaseType

dbGetFileDatabaseType(  
t_libName 
t_cellName 
t_viewName 
) 
=> t_dbFileDatabaseType

Description

Returns the t_libName, t_cellName, and t_viewName database type.

Arguments

t_libName

The library name.

t_cellName

The cell name.

t_viewName

The view name.

Value Returned

t_dbFileDatabaseType

The library/cell/view database type.

Examples

dbGetFileDatabaseType("newTech18" "M3xM2xG" "layout")
"OpenAccess"
