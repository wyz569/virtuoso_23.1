dbGetTrunkLongOffset2

dbGetTrunkLongOffset2(
d_trunkId 
)
=> f_longitudinalOffset2 / nil

Description

Returns the longitudinal offset2 value of the specified trunk pattern ID.

Arguments

d_trunkId

The trunk pattern ID whose longitudinal offset2 value must be returned.

Value Returned

f_longitudinalOffset2

  

The longitudinal offset2 value of the specified trunk pattern ID.

nil

The trunk pattern ID is incorrect.

Examples

dbGetTrunkLongOffset2(trunk1)
