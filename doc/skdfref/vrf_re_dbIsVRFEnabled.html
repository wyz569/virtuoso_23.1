dbIsVRFEnabled

dbIsVRFEnabled(
)
=> t / nil

Description

(Virtuoso MultiTech Framework) Confirms whether the Virtuoso RF Solution-related functionality is enabled in the current session.

This function will be deprecated in a future release.

Arguments

None.

Value Returned

t

Virtuoso RF Solution-related functionality is enabled.

nil

Virtuoso RF Solution-related functionality is not enabled.

Example

when(dbIsVRFEnabled()
  pts = dbGetCurvedPolygonPoints(css())
)
