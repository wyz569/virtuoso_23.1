deMapUserTypeToViewType

deMapUserTypeToViewType( 
t_userType 
) 
=> t_viewType

Description

Maps a user type to the dfII_viewType.

If the given user type is not found in the mapping, the function returns the user type itself.

Arguments

t_userType

User type.

Value Returned

t_viewType

DFII view type.

Example

deMapUserTypeToViewType("layout")
=>("masklayout")
