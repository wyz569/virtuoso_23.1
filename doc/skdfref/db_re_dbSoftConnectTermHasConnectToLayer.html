dbSoftConnectTermHasConnectToLayer

dbSoftConnectTermHasConnectToLayer ( 
d_termId 
) 
=> t / nil

Description

Checks if the given terminal has the softConnect connectToLayer attribute set.

If the given terminal is not a softConnect terminal, nil is returned.

Arguments

d_termId

The database ID of the terminal.

Value Returned

t

The terminal has the softConnect connectToLayer attribute set.

nil

The terminal is not a softConnect terminal or does not have the connectToLayer attribute set.

Examples

dbSoftConnectTermHasConnectToLayer( term )=> t
