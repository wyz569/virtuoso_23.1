Selection User Interface

Both selection by point and selection by area are supported directly using standard mouse bindings. Selection by point is accomplished by clicking the left mouse button, and selection by area is accomplished by drawing a selection rectangle with the left mouse button. The modes of selection are supported using modifier keys. The following table describes the standard selection mouse bindings.

Modifier Left Mouse Click Left Mouse Drag

None

Single-Select Point

Single-Select Box

Shift

Add-Select Point

Add-Select Box

Ctrl

Sub-Select Point

Sub-Select Box

Partial selection is controlled via function key F4. This key toggles partial selection on and off. Both partial point selection and partial area selection are affected. This key binding toggles the window environment variable partialSelect.
