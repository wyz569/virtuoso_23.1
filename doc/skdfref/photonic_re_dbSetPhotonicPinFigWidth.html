dbSetPhotonicPinFigWidth

dbSetPhotonicPinFigWidth(
d_pinFigId
n_value
)
=> t / nil

Description

(Virtuoso Photonics Option) Sets the given attribute value as the width of the specified photonic pinFig.

Arguments

d_pinFigId

Database ID of a photonics pinFig.

n_value

Width attribute value to be set.

Value Returned

t

Specified attribute value is set as width of the photonics pinFig.

nil

Width attribute value is not set.

Example

Set 5 as the width attribute value on the photonic pinFig pinFigId.

dbSetPhotonicPinFigWidth(pinFigId 5) 
=> t
