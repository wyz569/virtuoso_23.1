ddGetObjReadPathEx

ddGetObjReadPathEx(
b_objId
x_pathType
)
=> t_path / nil

Description

Returns the read path to b_objId. This function is similar to ddGetObjReadPath except that you can specify whether to look for the object in the temporary directory or the master directory with the x_pathType argument.

Arguments

b_objId

ID of the object.

x_pathType

One of the following integer values:

    • 0 Looks for the object only in the master directory
    • 1 Looks for the object only in the temporary directory
    • 2 Looks for the object in the temporary directory first; if it is not found, looks for it in the master directory. However, if the object is a library, returns the read path to the library itself (the master directory), not its temporary directory.

Value Returned

t_path

The read path to b_objId.

nil

The object was not found.
