dbSetRailDefFigType

dbSetRailDefFigType(
d_railDefId
t_railFigType
) 
=> t / nil

Description

Sets the railFigType for the specified railDef.

Arguments

d_railDef

Database ID of a railDef.

t_railFigType

Value of figType. Valid values are default, rect and pathSeg. Default value is rect.

Value Returned

t

Specified figType is set on the given railDef.

nil

Returned in case of failure.

Examples

Sets figType pathseg on rail definition d_rowRegionId.

dbSetRailDefFigType(d_rowRegionId "pathSeg")
