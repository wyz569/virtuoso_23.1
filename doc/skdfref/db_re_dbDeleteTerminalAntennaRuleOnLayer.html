dbDeleteTerminalAntennaRuleOnLayer

dbDeleteTerminalAntennaRuleOnLayer( 
d_termId 
t_modelName
t_dataName
tx_layer
)
=> t / nil

Description

Deletes the antenna rule of the terminal according to the specified antenna model name, antenna data name, and layer.

Arguments

d_termId

The database ID of the terminal.

t_modelName

Specifies the model name. Valid values: "defaultAntennaModel", "secondAntennaModel", "thirdAntennaModel", "fourthAntennaModel","antennaModel5" to "antennaModel32"

t_dataName

Specifies the data name. Valid values:

"ANTENNAGATEAREA"
"ANTENNADIFFAREA"
"ANTENNAPARTIALMETALAREA"
"ANTENNAPARTIALMETALSIDEAREA"
"ANTENNAPARTIALCUTAREA"
"ANTENNAMAXAREACAR"
"ANTENNAMAXSIDEAREACAR"
"ANTENNAMAXCUTCAR"

tx_layer

The layer name.

Value Returned

t

The antenna rule was removed successfully.

nil

An error occurred during execution.

Example

dbDeleteTerminalAntennaRuleOnLayer(
    termA "defaultAntennaModel" "ANTENNAGATEAREA" "M4")
