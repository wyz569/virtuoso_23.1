Path Attributes

A path is a geometry defined by the coordinates of its center line. A path has width, end and corner types, and variable extensions.

Attribute Modify Type Description
include Generic Shape Attributes.

beginExt

yes

float

Extension (in user units) of the path beyond the first vertex.

endExt

yes

float

Extension (in user units) of the path beyond the last vertex.

pathStyle

yes

string

"extendExtend"
"roundRound"
"truncateExtend"
"varExtendExtend"

points

yes

l_point

List of points for the path centerline (in user units).

width

yes

float

Width of the path (in user units).
