dbUnsetPhotonicPinFigRadius

dbUnsetPhotonicPinFigRadius(
pinFigId
)
=> t / nil

Description

(Virtuoso Photonics Option) Resets the attribute value of the radius for the specified photonics pinFig.

Arguments

pinFigId

Database ID of a photonics pinFig.

Value Returned

t

Radius attribute value of the specified photonic pinFig is reset.

nil

Radius attribute value is not reset.

Example

Radius attribute value of the photonic pinFig pinFigId is reset.

dbUnsetPhotonicPinFigRadius(pinFigId)
=> t
