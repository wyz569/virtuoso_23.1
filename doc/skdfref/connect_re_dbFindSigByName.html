dbFindSigByName

dbFindSigByName( 
d_cellView 
t_name 
) 
=> d_sig / nil

Description

Finds a signal in a cellview.

Arguments

d_cellView

Name of the cellview.

t_name

Name of the signal.

Value Returned

d_sig

The signal found.

nil

The signal is not found.

Example

sig = dbFindSigByName(cellview "VDD")
