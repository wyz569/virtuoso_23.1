Curvy Core Mathematical Object Attribute Query Functions

Curvy core objects such as connectors and curves can be queried using curvy core mathematical object attribute query functions. You can use these functions to check if the given curvy core curve is linear or to return the approximate length of a curve or a polycurve. You can also use these functions to return a point on a curve and to perform other attribute query operations on curvy core mathematical objects.
