Cross Section Viewer Functions

(Layout MXL Only) You can view cross sections of physical layers in a layout by using the cross section viewer. The viewer displays the different layers, from top to bottom, present in a cross section, enabling you to view each layer clearly. This is helpful to identify any missing or wrongly placed layers and to debug connection issues across layers.

You can use the following SKILL functions to work with the cross section viewer. Using these functions you can invoke the cross section viewer, load or unload the configuration file used for customizing the viewer, and specify the settings used for displaying the cross section of a layout.

  • geCrossSection
  • geCrossSectionAddLayerThickness
  • geCrossSectionAddLayerToSkip
  • geCrossSectionAddLayersToMerge
  • geCrossSectionAddMaterialToSkip
  • geCrossSectionClearConfig
  • geCrossSectionLoadConfig
