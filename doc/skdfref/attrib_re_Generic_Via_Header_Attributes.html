Generic Via Header Attributes

Finding Via Parameters

Via header attributes are common to via headers.

Attribute Modify Type Description
include Generic Object Attributes.

viaDef

no

objId

Database ID for the technology file of the viaDef associated with this viaHeader. The return value is nil if the viaHeader is not bound.

viaDefName

no

string

Cached string attribute which returns the name of the associated viaDef. Usually available even if the viaHeader is not bound.

master

no

objId

ID of the cellview that is acting as the master of the viaHeader. Can be nil if the viaHeader is not bound.

subHeaders

no

l_objId

List of database IDs that identifies sub-via-Headers. The attribute value is nil if the viaHeader being queried is not a super-viaHeader.

vias

no

l_objId

Database ID of the vias associated with the viaHeader.

overrideParams

no

l_params

List of override parameters and values if they exist.
