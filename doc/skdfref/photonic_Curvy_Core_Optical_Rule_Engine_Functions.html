Curvy Core Optical Rule Engine Functions

The SKILL functions associated with the optical rule engine (ORE) can be used to evaluate ORE expressions, compile the ORE expressions specified as a file or a list, or retrieve a list of ORE context names. You can also use the ORE SKILL functions to get ORE contexts or to purge specific or all contexts.
