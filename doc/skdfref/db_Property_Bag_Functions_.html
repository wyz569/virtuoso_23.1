Property Bag Functions

The property bag is a database object that contains only properties and is identified by dbBagId. This topic provides a list of Cadence® SKILL functions associated with the property bag. They can be used to create or modify properties in the property bag.

Only the functions listed here are supported for public use. All other functions, regardless of their name or prefix, and undocumented aspects of the functions described below, are private and are subject to change at anytime.
