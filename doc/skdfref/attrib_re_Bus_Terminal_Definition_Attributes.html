Bus Terminal Definition Attributes

Attribute Modify Description
Include Generic Object Attributes.

baseName

no

string

Returns base name for the busTermDef.

bitOrder

no

integer

Returns bit order for the busTermDef. If the busTermDef is implicit, “none” will be returned.

minIndex

no

integer

Returns smallest bit number in use for the busTermDef.

maxIndex

no

integer

Returns largest bit number in use for the busTermDef.

numBits

no

integer

Returns the number of bits covered for this busTermDef.

numBitsUsed

no

integer

Returns the number of busTermBits used for the busTermDef.
