geSetMaxCreatedProbe

geSetMaxCreatedProbe( 
x_maxCreateProbe 
)
=> t / nil

Description

Sets the maximum number of created probes.

Arguments

x_maxCreateProbe

Maximum number of probes.

Value Returned

t

The maximum limit of probes has been set.

nil

The maximum number of probes could not be set.

Examples

Sets the maximum number of probes to 1000.

geSetMaxCreatedProbe(1000)
