dbGetCellViewPlaceArea

dbGetCellViewPlaceArea(
d_cellviewId 
)
=> l_customPlacementAreas / nil

Description

Returns a list of custom placeAreas in the specified cellview.

Arguments

d_cellviewIdy

Database ID of the cellview.

Value Returned

l_customPlacementAreas

  

A list of custom placeAreas is returned.

nil

The cellview ID is invalid or no custom placeAreas exist.

Examples

Returns a list of custom placeAreas.

myCustomPlaceAreas = dbGetCellViewPlaceArea(geGetEditCellView())
