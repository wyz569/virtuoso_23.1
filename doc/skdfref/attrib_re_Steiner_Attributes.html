Steiner Attributes

Attribute Modify Type Description
include Generic Object Attributes.

bBox

yes

bBox

bBox associated with the specified steiner

connRoutes

no

l_objId

A list of route IDs that connected to the steiner.

layer

yes

string

Number of layer associated with the Steiner.

layerHeader

no

objId

Database ID of the layer header associated with the Steiner.

Steiner attributes can be accessed as follows:

steinerId~> attributeName
steinerPinId~> attributeName

Steiner pin attributes can be modified as follows:

steinerPinId~> attributeName = attributeValue

Example:

steinerId~>layer => t_layerNum 
steinerId~>layer = tx_layerNum => t
