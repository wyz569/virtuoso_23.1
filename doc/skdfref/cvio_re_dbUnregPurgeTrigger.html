dbUnregPurgeTrigger

dbUnregPurgeTrigger( 
s_triggerFunc 
) 
=> t / nil

Description

Unregisters a user-defined trigger function.

Arguments

s_triggerFunc

User-defined trigger function.

Values Returned

t

The trigger function gets unregistered successfully.

nil

The un-registration fails.

Examples

dbUnregPurgeTrigger(’myTrigFunc)
