dbGetTopologyPatternType

dbGetTopologyPatternType(
d_patternId 
)
=> t_tpGroupType / nil

Description

Returns the type of the specified topology pattern ID. The valid types are topology, trunkConnect, trunk, twig, strap, and generic.

Arguments

d_patternId

The topology pattern ID whose group type must be returned.

Value Returned

t_tpGroupType

The type of the topology pattern associated with the specified topology pattern ID.

nil

The specified topology pattern ID is incorrect.

Examples

dbGetTopologyPatternType(top)
=>"topology"
dbGetTopologyPatternType(trunkConn)
=>"trunkConnect"
dbGetTopologyPatternType(trunk)
=>"trunk"
dbGetTopologyPatternType(twig)
=>"twig"
dbGetTopologyPatternType(strap)
=>"strap"
