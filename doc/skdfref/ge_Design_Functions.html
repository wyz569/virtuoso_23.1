Design Functions

A design can be opened for reading or editing. Whether a design is opened in edit or read mode, all designs under the hierarchy of that design are opened in read mode. The only exception is edit-in-place, where both the top design and the design being edited can be modified; the other designs maintain their non-modifiable status.

After a design is opened, you can change the mode from read to edit. If a design has already been modified, changing the mode from edit to read makes the system to prompt you to save or discard the changes made. If you choose the discard option, the changes are ignored and the original design is brought back from the disk. You cannot make a cell editable if you do not have write permission. The name of the current design and its mode are displayed in the window banner.

You can use the following SKILL functions to work with your designs:

geChangeCellView

geChangeEditMode

geDiscardEdits

geNewWindow

geReturn

geReturnToLevel

geSave

geSaveAs

geSaveHier
