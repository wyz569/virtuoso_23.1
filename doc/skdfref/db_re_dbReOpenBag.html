dbReOpenBag

dbReOpenBag( 
g_bagId 
t_mode 
) 
=> t / nil

Description

Reopens the given g_bagId for the given mode.

Arguments

g_bagId

g_bagId to be reopened.

t_mode

Mode in which the property bag will be reopened.

Value Returned

t

The bag is reopened in the desired mode.

nil

It fails to reopen the bag.

Examples

dbReOpenBag( bagId "a" )
