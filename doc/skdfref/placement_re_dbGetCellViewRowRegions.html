dbGetCellViewRowRegions

dbGetCellViewRowRegions(
d_cellviewId
)
=> l_rowRegionSpecs / nil

Description

Returns a list of rowRegions in the specified cellview.

Arguments

d_cellviewId

Database ID of a cellview.

Value Returned

l_rowRegionSpecs

List of database IDs of rowRegion in the given cellview.

nil

Returned in case of failure.

Examples

Returns a list of rowRegions in the cellview cv.

dbGetcellviewRowRegion(cv)
