dbGetCompTypeSetDefCompFilters

dbGetCompTypeSetDefCompFilters(
d_compDefId
)
=> l_compFilters / nil

Description

Returns the component filters in the specified compTypeSetDef.

Arguments

d_compDefID

Database ID of a compTypeSetDef.

Value Returned

l_compFilters

List of component filters.

nil

Returns an error in case of failure.

Examples

Creates a compTypeSetDef and retrieves its compFilters.

; Create compTypeSetDef
         compAttr = list(nil)
         compAttr->refOnArea  = "centerAlignment"
         compAttr->refOnComp  = "anyAlignment"
         compAttr->allowedTypes = list("compType1" "compType2")
         compAttr->orients = list("R0")
         compAttr->compFilters = list(list("cellLib" "nf*" "layout") list("cellLib" "ng*" "layout"))
         cdef = dbCreateCompTypeSetDefByAttr(cv "compTypeSetDef" compAttr);
         ;; Get compFilters
         dbGetCompTypeSetDefCompFilters(cdef)
