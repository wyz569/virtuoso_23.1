ccIsEnabled

ccIsEnabled(
)
=> t / nil

Description

(Virtuoso Photonics Option) Checks if the CurvyCore engine is enabled.

Arguments

None

Value Returned

t

The CurvyCore engine is enabled.

nil

The CurvyCore engine is not enabled.

Examples

ccIsEnabled()
=> nil
ccCreateCurve("t" "1" 0:1)
=> *Error* (CC-200007): CurvyCore component is not initialized.
