Copying and Renaming Objects with Design Intents

To copy an object including any associated design intent:

  1. From the view, choose Edit – Copy.
  2. Press F3 to display the Copy form.
  3. Select the Include Design Intent option.
  4. Select the object you want to copy.
  5. Click a destination for the copy.

Objects with design intents associated can be renamed as normal. The new name is reflected in any design intent associated with the object when the Check and Save command is run.
