get_filename

get_filename( 
p_port 
) 
=> s_result

Description

Returns the file name of a port.

Arguments

p_port

A port object.

Value Returned

x_result

The file name of the port.

Examples

aPort                 
=> port:"inFile"
get_filename( aPort ) 
=> "inFile"
