range

range( 
n_num1 
n_num2 
) 
=> l_result

Description

Returns a list whose first element is n_num1 and whose tail is n_num2. This function is a prefix form of the : operator.

Arguments

n_num1

First element of the list.

n_num2

Tail of the list.

Value Returned

l_result

Result of the operation.

Examples

L = range(1 2) 
=> (1 2)
car(L) 
=> 1
cdr(L) 
=> (2)
L = range(1.1 3.3) 
=> (1.1 3.3)
car(L) 
=> 1.1
cdr(L) 
=> (3.3)
