charToInt

charToInt( 
s_char 
) 
=> x_ascii

Description

Returns the ASCII code of the first character of the given symbol. In SKILL, a single character symbol can be used as a character value.

Arguments

s_char

A symbol.

Value Returned

x_ascii

The ASCII code of the (first) character of the given symbol.

Examples

charToInt('B)
=> 66
charToInt('Before)
=> 66
