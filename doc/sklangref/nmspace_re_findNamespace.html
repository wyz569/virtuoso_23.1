findNamespace

findNamespace( 
t_name 
) 
=> o_namespace / nil

Description

Returns the namespace object with the specified name.

Arguments

t_name

Specify the name for which you want to retrieve the namespace object.

Value Returned

o_namespace

The namespace object.

nil

No namespace object exists with the given name.

Examples

findNamespace("A")
=> ns@A
