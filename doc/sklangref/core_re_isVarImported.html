isVarImported

isVarImported( 
s_var 
) 
=> t / nil

Description

Checks if the specified variable was imported into SKILL++ or not.

Arguments

s_var

The variable to be checked.

Value Returned

t

The specified variable s_var was imported into SKILL++.

nil

If the given variable is not imported.

Examples

isVarImported('myvar)
=> nil
