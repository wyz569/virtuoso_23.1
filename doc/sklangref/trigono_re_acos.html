acos

acos( 
n_number 
) 
=> f_result

Description

Returns the arc cosine of a floating-point number or integer.

Arguments

n_number

Floating-point number or integer.

Value Returned

f_result

Arc cosine of n_number.

Examples

acos(0.3)
=> 1.266104
