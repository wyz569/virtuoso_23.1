popf

popf( 
g_place
) 
=> g_result

Description

A pop that uses the setf function. It returns the value for g_place that is removed.

Arguments

g_place

Place to be modified.

Value Returned

g_result

The value for g_place that is removed.

Examples

a = '((4 1) 2 3)
popf(car(a) )
=> 4
a == '((1) 2 3)
