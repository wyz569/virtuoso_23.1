minusp

minusp( 
g_general 
) 
=> t / nil

Description

Checks if a value is a negative number. Same as negativep.

Arguments

g_general

Number to check.

Value Returned

t

If g_general is a negative number.

nil

If g_general is not a negative number.

Examples

minusp( 3 )    
=> nil
minusp( -3 )   
=> t
