exp

exp( 
n_number 
) 
=> f_result

Description

Raises e to a given power.

Arguments

n_number

Power to raise e to.

Value Returned

f_result

Value of e raised to the n_numberth power.

Examples

exp( 1 )  
=> 2.718282
exp( 3.0) 
=> 20.08554
