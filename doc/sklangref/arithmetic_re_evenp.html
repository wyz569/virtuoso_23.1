evenp

evenp( 
g_general 
) 
=> t / nil

Description

Checks if a number is an even integer.

Arguments

g_general

Number to check.

Value Returned

t

If g_general is an even integer.

nil

If g_general is not an even integer.

Examples

evenp( 59 )
=> nil
evenp( 60 )
=> t
evenp( 2.0 )
=> nil             ; Number is even, but not an integer.
