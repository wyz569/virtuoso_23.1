atom

atom( 
g_arg 
) 
=> t / nil

Description

Checks if an object is an atom.

Atoms are all SKILL objects except non-empty lists. The special symbol nil is both an atom and a list.

Arguments

g_arg

Any SKILL object.

Value Returned

t

If g_arg is an atom.

nil

If g_arg is not an atom.

Examples

atom( 'hello )  
=> t
x = '(a b c)
atom( x )       
=> nil
atom( nil )     
=> t
