booleanp

booleanp( 
g_obj 
) 
=> t / nil

Description

Checks if an object is a boolean. Returns t if the object is t or nil. Returns nil otherwise.

Arguments

g_obj

Any SKILL object.

Value Returned

t

If g_obj is either t or nil.

nil

If g_obj is not a boolean.

Examples

(booleanp 0 ) 
=> nil
(booleanp nil) 
=> t
(booleanp t) 
=> t
