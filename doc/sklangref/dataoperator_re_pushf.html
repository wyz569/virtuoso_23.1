pushf

pushf( 
g_obj
g_place
) 
=> g_newPlaceValue

Description

A push that uses the setf function. It modifies the contents of the original storage location.

Arguments

g_obj

New value to be pushed.

g_place

Place to be modified with the new value.

Value Returned

g_newPlaceValue

New value.

Examples

a = list((list 1)  2 3)
pushf(4 (car a))
=> a == '((4 1) 2 3)
