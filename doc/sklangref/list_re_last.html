last

last( 
l_arg 
) 
=> l_result

Description

Returns the last list cell in a list.

Arguments

l_arg

List of elements.

Value Returned

l_result

Last list cell (not the last element) in l_arg.

Examples

last( '(a b c) )         
=> (c)
z = '(1 2 3)
last(z)                  
=> (3)
last( '(a b c (d e f)))  
=> ((d e f))
