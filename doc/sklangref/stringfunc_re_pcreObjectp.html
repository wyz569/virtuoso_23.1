pcreObjectp

pcreObjectp( 
g_arg 
) 
=> t / nil

Description

Checks to see whether the given argument is a pcreObject or not.

Arguments

g_arg

A value to be checked.

Value Returned

t

g_arg is a pcreObject.

nil

g_arg is not a pcreObject.

Examples

a = pcreCompile("abc[0-9]+")
=> pcreobj@0x83b8018
(pcreObjectp a)
=> t
(pcreObjectp 9)
=> nil
