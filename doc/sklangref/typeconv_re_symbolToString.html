symbolToString

symbolToString( 
s_symbolName 
) 
=> t_string

Description

Converts a symbol to a string of the same name. This function is the same as get_pname.

Arguments

s_symbolName

Symbol to convert.

Value Returned

t_string

String with the same name as the input symbol.

Examples

y = symbolToString( 'test2)
=> "test2"
sprintf(nil "%L" y)
=> "\"test2\""
