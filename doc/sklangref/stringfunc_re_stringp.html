stringp

stringp( 
g_value 
) 
=> t / nil

Description

Checks if an object is a string.

The suffix p is usually added to the name of a function to indicate that it is a predicate function.

Arguments

g_value

A data object.

Value Returned

t

g_value is a string.

nil

g_value is not a string..

Examples

stringp( 93)
=> nil
stringp( "93")
=> t
