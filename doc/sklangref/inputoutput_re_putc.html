putc

putc(
x_symbol
p_port
)
=> s_symbol

Description

Puts the x_symbol to p_port (to complement getc function)

Arguments

x_symbol

Symbol number

p_port

An output port

Value Returned

s_symbol

The symbol that was put

Examples

putc(1  poport)
=> \001
