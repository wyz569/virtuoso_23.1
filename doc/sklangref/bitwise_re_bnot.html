bnot

bnot( 
x_op 
) 
=> x_result

Description

Returns the integer result of the Boolean NOT operation on each parallel pair of bits in each operand. Prefix form of the ~ (one’s complement) unary operator.

Arguments

x_op

Operand to be evaluated.

Value Returned

x_result

Result of the operation.

Examples

bnot(12)  
=> -13
bnot(-12) 
=> 11
