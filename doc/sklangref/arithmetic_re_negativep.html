negativep

negativep( 
n_num 
) 
=> t / nil

Description

Checks whether the specified value is a negative number. Same as minusp.

Arguments

n_num

Number to check.

Value Returned

t

n_num is a negative number.

nil

n_num is not a negative number.

Examples

negativep( 3 )    
=> nil
negativep( -3 )   
=> t
