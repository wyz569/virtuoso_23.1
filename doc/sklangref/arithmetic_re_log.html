log

log( 
n_number 
) 
=> f_result

Description

Returns the natural logarithm of a floating-point number or integer.

Arguments

n_number

Floating-point number or integer.

Value Returned

f_result

Natural logarithm of the value passed in.

If the value of n_number is not a positive number, an error is signaled.

Examples

log( 3.0 ) 
=> 1.098612
