make_<name>

make_<name>( 
... 
) 
=> r_defstruct

Description

Creates an instance of a defstruct specified by name.

Arguments

...

Initial values for structure elements (slots).

Value Returned

r_defstruct

Copy of the given instance

Examples

defstruct(myStruct a b c) 
=> t
m1 = make_myStruct(?a 3 ?b 2 ?c 1) 
=> array[5]:3436504
m2 = copy_myStruct(m1) 
=> array[5]:3436168
