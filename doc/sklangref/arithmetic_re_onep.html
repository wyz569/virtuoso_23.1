onep

onep( 
g_value 
) 
=> t / nil

Description

Checks whether the given object is equal to one. onep is a predicate function.

Arguments

g_value

A SKILL object that is either a floating-point number or an integer.

Value Returned

t

If g_value is equal to one.

nil

If g_value is not equal to one.

Examples

onep( 1 ) 
=> t
onep( 7 ) 
=> nil
onep( 1.0 ) 
=> t
