Doc Assistant - Online Mode

If you are connected to the Internet and have a valid license, Doc Assistant is launched in Online mode by default. In this mode, search results are displayed from the documents that are available on the Doc Assistant server and Cadence Support.

You can also access Cadence Support content, such as App Notes, RAKs, and Troubleshooting related information in this mode.   

If Doc Assistant is in Offline mode, you can switch to Online mode by switching on the View Documentation Online option from the hamburger menu.
