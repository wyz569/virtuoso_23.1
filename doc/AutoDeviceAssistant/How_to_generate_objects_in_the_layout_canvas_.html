Product Version IC23.1, September 2023

How to generate objects in the layout canvas?

To generate objects in the layout canvas, on the Initialize tab of the Auto P&R assistant:

  1. Select Generate to enable the device generation options.
  2. Select which devices to generate: Instances, Pins, and Boundary.
  3. With Pins selected, select the pin layer that contains the pins to be generated.
  4. With Boundary selected, select one of the following combinations:
    • Utilization and Aspect Ratio
    • Width and Height
  5. Select Preserve to enable the options to preserve existing components.
  6. Select the components to be preserved: Instances, Boundary, Constraints, Pins, and Rows.
  7. Select Use Reference Layout to enable the options in the Reference Layout section.
  8. Click Browse to select a reference cellview. Objects in the current cellview are placed as per their positions in the reference cellview.
  9. Select the components to be placed as per their reference: Instance Positions, Pins, Boundary, and Constraints.
  10. Select Extract Templates to extract Modgen templates from the reference layout cellview.
  11. Select Update PCell Params to update the parameters of the specified technology file.
  12. Click Apply to generate the selected objects in the layout canvas.
