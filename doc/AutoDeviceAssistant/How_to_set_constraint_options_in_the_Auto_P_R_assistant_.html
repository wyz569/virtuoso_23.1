How to set constraint options in the Auto P&R assistant?

To set constraint options in the Auto P&R assistant:

  1. Open the Constraints tab.
  2. Select Ignore Self-symmetry Constraints of Instances to specify that instance self-symmetry constraints are to be ignored.

  3. In the Groups and Constraints pane of the Constraints tab

  4. Uncheck the finders you do not want to run. The sequence of the finders determines the sequence in which constraints are listed. You can reorder the finders by dragging them to the desired locations within the list before generating them.

  5. Click Find Structures to search for all matching structures in the source cellview following the order in which they are listed.

    All structures and device groups found are listed under their respective categories in the Groups and Constraints pane.
