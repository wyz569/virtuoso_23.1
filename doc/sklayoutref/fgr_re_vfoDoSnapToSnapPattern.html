vfoDoSnapToSnapPattern

vfoDoSnapToSnapPattern (
d_inst
)
=> t / nil

Description

Snaps the fluid object to the snap pattern.

Arguments

d_inst

The ID of the fluid object instance.

Value Returned

t

The fluid object was snapped to the snap pattern.

nil

The fluid object could not be snapped to the snap pattern.

Example

vfoDoSnapToSnapPattern(objFluid)

In the above example, objFluid is an instance of the fluid object.
