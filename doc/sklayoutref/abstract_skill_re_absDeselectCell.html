absDeselectCell

absDeselectCell(
name
)
=> 0 / 1

Description

Deselects a named cell.

Arguments

name

Specifies the name of the cell to be deselected.

Value Returned

0

Indicates an error.

1

Indicates successful operation.

Examples

Deselects cell1.

absDeselectCell "cell1"
=> 1
