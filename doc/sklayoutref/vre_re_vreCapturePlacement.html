vreCapturePlacement

vreCapturePlacement(
)
=> t / nil

Description

Captures placement from the current layout view.

Arguments

None

Value Returned

t

Placement was captured.

nil

The command was unsuccessful.

Examples

The following example captures placement from the active layout view.

vreCapturePlacement()
=> t
