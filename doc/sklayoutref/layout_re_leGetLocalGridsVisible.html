leGetLocalGridsVisible

leGetLocalGridsVisible(
)
=> t / nil

Description

Returns the visibility setting of local snap pattern grids.

Arguments

None

Value Returned

t

The local snap pattern grid is visible.

nil

The local snap pattern grid is not visible.

Example

Returns t if the local snap pattern grid is visible.

leGetLocalGridsVisible()
