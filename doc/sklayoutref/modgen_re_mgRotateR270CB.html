mgRotateR270CB

mgRotateR270CB(
)
=> t / nil

Description

Rotates the selected Modgen around its center using the R270 transform.

Arguments

None

Value Returned

t

The Modgen is rotated.

nil

The Modgen is not rotated.

Example

Rotates the selected Modgen.

mgRotateR270CB()
