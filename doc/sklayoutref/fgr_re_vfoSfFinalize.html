vfoSfFinalize

vfoSfFinalize(
pcell vfoSfPCellClass
) 
=> t / nil

Description

Performs any required cleanup.

Arguments

pcell

SKILL++ object of vfoSfPcellClass.

Value Returned

t

The cleanup was performed successfully.

nil

The cleanup could not be performed.

Example

vfoSfFinalize(obj)
    => nil
