viaSaveViaVariants

viaSaveViaVariants(
d_cellViewId
) 
=> t

Description

Saves the via variants from the Create Via command into the specified cellview.

Arguments

d_cellViewId

The cellview ID in which the via variants are saved.

Value Returned

t

Returns t if via variants are saved.

Example

viaSaveViaVariants(cvId)
