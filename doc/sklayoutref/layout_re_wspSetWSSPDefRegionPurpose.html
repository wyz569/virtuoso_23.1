wspSetWSSPDefRegionPurpose

wspSetWSSPDefRegionPurpose (
d_wsspDefId 
t_purposeName
) 
=> t / nil

Description

Overrides the region purpose attribute for the specified widthSpacingSnapPatternDefs (WSSPDef).

Arguments

d_wsspDefId

Database ID of the WSSPDef.

t_purposeName

The region purpose name.

Value Returned

t

The region purpose attribute is overridden.

nil

The region purpose attribute is not overridden.

Example

wspSetWSSPDefRegionPurpose(ldef "localWSP")
