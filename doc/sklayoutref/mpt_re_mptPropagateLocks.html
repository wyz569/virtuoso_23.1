mptPropagateLocks

mptPropagateLocks(
[ d_cellviewID ]
[ g_convertGroups ] 
)
=> t / nil

Description

Propagates all the locks visible from the top level to connected shapes through the hierarchy.

Arguments

d_cellviewID

The database ID of the top-level cellview. If this argument is not given, the current edit cellview is used.

g_convertGroups

When set to t, all shapes that are in a color group but are not locked will be locked. This is the default.
When set to nil, lock propagation on color groups is not performed.

Values Returned

t

Lock propagation was successful.

nil

Lock propagation was not successful.

Example

mptPropagateLocks()

Propagates locks for the current edit cellview.
