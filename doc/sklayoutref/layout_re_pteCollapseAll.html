pteCollapseAll

pteCollapseAll(
[ t_panelName ]
[ w_windowId ]
)
=> t / nil

Description

Collapses all items in the specified Palette panel.

Arguments

t_panelName

Name of the Palette panel.
Valid values: Layers, Objects, and Grids
Default: Currently active panel; if no panel is active, all items in the Layers panel are collapsed

w_windowId

ID of the window containing the Palette in which you want to collapse items.
Valid values: ID of any open window
Default: Current window

Value Returned

t

All items in the specified panel were collapsed.

nil

The command was unsuccessful.

Examples

pteCollapseAll("Grids")
pteCollapseAll("Objects" window(2))
