cphGetIgnoreForCheck

cphGetIgnoreForCheck(
g_physConfigID
t_path
)
=> t / nil

Description

Returns the effective value of the Ignore for check attribute for the specified occurrence in the specified physical configuration.

Arguments

g_physConfigID

ID of the physical configuration cellview.

t_path

Path to the specific occurrence of an instance.

Value Returned

t

The specified occurrence will be ignored for check.

nil

The specified occurrence will not be ignored for check.

Example

val=cphGetIgnoreForCheck(physConfigID "I0/N0")
