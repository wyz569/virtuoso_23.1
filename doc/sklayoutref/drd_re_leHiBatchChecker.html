leHiBatchChecker

leHiBatchChecker( 
[ w_winID ] 
) 
=> t / nil

Description

Opens the Batch Checker form.

Batch-Check mode is available only in Layout XL and higher tiers. For more information, see Batch Checking Mode in Virtuoso Design Rule Driven Editing User Guide.

Argument

w_winID

ID of the window in which you want to open the Batch Checker form.

Value Returned

t

The Batch Checker form was opened.

nil

The Batch Checker form was not opened.

Examples

leHiBatchChecker()
leHiBatchChecker(window(3))
