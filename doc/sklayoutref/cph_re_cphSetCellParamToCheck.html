cphSetCellParamToCheck

cphSetCellParamToCheck(
g_physConfigID
t_logLib
t_logCell
t_listOfParamNames
)
=> t / nil

Description

Defines the list of parameter names to be checked when running Check Against Source on the specified logical cell in the specified physical configuration.

Arguments

g_physConfigID

ID of the physical configuration cellview.

t_logLib

Name of the library containing the logical cell.

t_logCell

Name of the logical cell.

t_listOfParamNames

List of parameter names to be checked.

Separate each parameter name with a space and enclose the list in quotation marks.

Value Returned

t

The list of parameter names to be checked was set.

nil

The list of parameter names to be checked was not set.

Example

val=cphSetCellParamToCheck(physConfigID "cph" "nand" "l w")
