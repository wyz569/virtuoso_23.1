leToggleDrdMode

leToggleDrdMode(
)
=> t

Description

Cycles through DRD modes: Off, Notify, Enforce, and Enforce and Notify.

Arguments

None

Value Returned

t

DRD mode was reset to one of the possible values.

Example

leToggleDrdMode()
(LE-102620): Change drdMode from Notify to Enforce...
t
