apDeleteTrims

apDeleteTrims (
)
=> t / nil

Description

(Virtuoso Layout Suite EXL and above) Deletes trims from the current cellview.

Arguments

None

Value Returned

t

Trims were deleted.

nil

Trims were not deleted.

Examples

Deletes trims from the current cellview.

apDeleteTrims()
