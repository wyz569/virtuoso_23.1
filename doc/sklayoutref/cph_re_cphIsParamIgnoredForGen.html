cphIsParamIgnoredForGen

cphIsParamIgnoredForGen(
g_physConfigID
t_path
t_paramName
)
=> t / nil

Description

Returns whether the specified parameter on the specified occurrence is ignored for generation.

Arguments

g_physConfigID

ID of the physical configuration cellview.

t_path

Path to the specific occurrence of an instance.

t_paramName

Name of the parameter.

Value Returned

t

The specified parameter is ignored for generation.

nil

The specified parameter is not ignored for generation.

Example

cphIsParamIgnoredForGen(physConfigID "I0/N0" "m")
