mgAddDummyRCLeftCB

mgAddDummyRCLeftCB(
)
=> t / nil

Description

Adds a complete row of dummy devices to the left of the selected instances.

Arguments

None

Value Returned

t

The row of dummy devices was added.

nil

The row of dummy devices was not added.

Example

Adds a row of dummies to the left of the selected instances.

mgAddDummyRCLeftCB()
