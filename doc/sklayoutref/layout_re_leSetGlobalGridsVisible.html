leSetGlobalGridsVisible

leSetGlobalGridsVisible(
g_isVisible
)
=> t / nil

Description

Turns on or off the visibility of global snap pattern grids.

Arguments

g_isVisible

Boolean specifying whether the global snap pattern grid is visible. Valid Values: t for visible or nil for invisible

Value Returned

t

The global snap pattern grid is visible.

nil

The global snap pattern grid is not visible.

Example

Returns t if global snap pattern grid is set to visible and nil if it is not.

leSetGlobalGridsVisible(t)
