Deleted Custom Digital Placer SKILL Functions

The following Cadence® SKILL functions associated with Virtuoso® Custom Digital Placer are no longer supported in the current release and will be ignored if called. They will be removed completely in the next major release.

  • lxPlcAppendPlaceSetupFieldValue
  • lxPlcGetPlaceSetupFieldChoices
  • lxPlcGetPlaceSetupFieldItems
  • lxPlcGetPlaceSetupFieldValue
  • lxPlcIsPlaceSetupFieldEditable
  • lxPlcIsPlaceSetupFieldEnabled
  • lxPlcReplacePlaceSetupField
  • lxPlcSetPlaceSetupFieldChoices
  • lxPlcSetPlaceSetupFieldEditableState
  • lxPlcSetPlaceSetupFieldEnableState
  • lxPlcSetPlaceSetupFieldItems
  • lxPlcSetPlaceSetupFieldValue
  • lxTpTemplateLoadAll
  • vcpfeCreateEstimatedRows

Use the Placement Planning form and corresponding environment variables to set the default values in the form.
  • vcpPlaceBoundaryCells

This function is no longer supported and will be ignored if specified. It has been retained to maintain backward compatibility.
