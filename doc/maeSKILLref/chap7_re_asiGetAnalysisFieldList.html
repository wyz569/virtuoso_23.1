asiGetAnalysisFieldList

asiGetAnalysisFieldList( 
o_analysis 
)
=> l_fieldList / nil

Description

Returns a list of analysis field objects defined for a particular analysis object.

Arguments

o_analysis

Specifies an analysis object.

Value Returned

l_fieldList

Returns the list of field objects for the specified analysis object.

nil

The analysis has no fields associated with it.

Examples

The following example returns the list of DC analysis fields:

dcAnal = asiGetAnalysis(session 'dc)
asiGetAnalysisFieldList( dcAnal )
