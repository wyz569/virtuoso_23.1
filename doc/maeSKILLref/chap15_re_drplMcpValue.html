drplMcpValue

drplMcpValue( 
s_probe 
) 
=> f_power

Description

Returns the main channel power value.

Arguments

s_probe

The name of the probe for which the main channel power needs to be calculated.

Value Returned

f_power

The main channel power value.

Examples

Returns the main channel power for the WPRB0 probe:

drplMcpValue('WPRB0) 
=> 6.87651
