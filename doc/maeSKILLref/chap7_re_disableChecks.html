disableChecks

disableChecks(
t_check1 
[ t_check2 t_check3... ]
)
=> l_checks / nil

Description

Disables device checks. Only enabled device checks appear in Spectre input deck.

This function can be used only in an Analog Design Environment L session. When the enableDeviceChecking environment variable is set to t, the function can be used for an Analog Design Environment XL session too.

Argument

t_check1 - t_checkn

Names of  checks to be disabled.

Value Returned

l_checks

Returns list of checks passed.

nil

Returns nil and prints an error if the simulator is not Spectre.

Example

disableChecks("check1" "check2" "check3")
