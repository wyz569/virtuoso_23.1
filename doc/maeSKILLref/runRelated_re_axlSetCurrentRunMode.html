axlSetCurrentRunMode

axlSetCurrentRunMode( 
x_hsdb 
t_mode 
) 
=> t / nil

Description

Sets the current simulation run mode for the given database.

Arguments

x_hsdb

Setup database handle.

t_mode

Run mode.

Valid Values for ADE Explorer:

  • "Single Run, Sweeps and Corners"
  • "Monte Carlo Sampling"

Valid Values for ADE XL and ADE GXL:

  • "Single Run, Sweeps and Corners"
  • "Monte Carlo Sampling"
  • "Global Optimization"
  • "Local Optimization"
  • "Improve Yield"
  • "High Yield Estimation"
  • "Sensitivity Analysis"

Valid Values for ADE Assembler:

  • "Single Run, Sweeps and Corners"
  • "Monte Carlo Sampling"
  • "Global Optimization"
  • "Local Optimization"
  • "Improve Yield"
  • "High Yield Estimation"
  • "Sensitivity Analysis"
  • "Feasibility Analysis"
  • "Worst Case Corners"
  • "Manual Tuning"
  • "Size Over Corners"
  • "Run Plan"
  • "Fault Simulation"
  • Feasibility Analysis"
  • "Worst Case Corners"
  • "Manual Tuning"
  • "Size Over Corners"
  • "Manual Optimization
  • "Conjugate Gradient Optimization"
  • "Feasibility Analysis"
  • "Worst Case Corners"
  • "Manual Tuning"
  • "Size Over Corners"
