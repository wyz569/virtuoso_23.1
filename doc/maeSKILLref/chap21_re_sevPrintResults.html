sevPrintResults

sevPrintResults(
t_session
t_type
)
=> t / nil

Description

Displays the Print Results form that lets you print the selected type of simulation results.

Arguments

t_session

The simulation environment session.

t_type

The type of results to print. Possible values are: 'dcOpPoints, 'tranOpPoints, 'modelParameters, 'sensitivities, 'noiseParameters, 'noiseSummary, 'dcNodeVoltages, or 'tranNodeVoltages.

Value Returned

t

The call is successful.

nil

The call is unsuccessful.
