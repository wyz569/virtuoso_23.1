drplPacVolGnExpDen

drplPacVolGnExpDen(
t_denSigStr
rh
t_name
)
=> o_waveform / nil

Description

Plots the PAC voltage wave.

Arguments

t_denSigStr

The defined signal string.

rh

The reference harmonic.

t_name

Name.

Value Returned

o_waveform

Returns the waveform of PAC voltage.

nil

Returns nil, otherwise.

Examples

drplPacVolGnExpDen("v(\"/VLO\" ?result \"hbac\")" '(0) nil)
