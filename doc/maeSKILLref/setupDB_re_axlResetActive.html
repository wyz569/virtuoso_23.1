axlResetActive

axlResetActive( 
x_hsdb 
)
=> t / nil

Description

Resets the active setup database.

Arguments

x_hsdb

Setup database handle

Value Returned

t

The active setup database is reset.

nil

The active setup database is not reset.

Examples

Resets the setup database with handle 1003.

axlResetActive( 1003 )
=> t
