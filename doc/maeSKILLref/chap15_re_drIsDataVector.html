drIsDataVector

drIsDataVector( 
g_value
)
=> t / nil

Description

Returns t when g_value is a valid drVector data object and nil if it is not one.

Arguments

g_value

Value checked.

Value Returned

t

Returns t when g_value is a drVector data object.

nil

Returns nil if g_value is not a drVector data object.

Examples

wave = drCreateEmptyWaveform()
drIsDataVector( wave )
=> nil
