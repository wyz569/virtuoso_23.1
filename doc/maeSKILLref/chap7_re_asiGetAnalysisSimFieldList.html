asiGetAnalysisSimFieldList

asiGetAnalysisSimFieldList(
o_session 
o_ana 
) 
=> l_simFieldObjList

Description

Gets a list of simulator analysis field objects that need to be netlisted.

Arguments

o_session

The simulation session object.

o_ana

The analysis object.

Value Returned

l_simFieldObjList

The list of simulator field objects.
