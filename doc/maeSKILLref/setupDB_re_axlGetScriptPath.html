axlGetScriptPath

axlGetScriptPath( 
x_script 
) 
=> t_path / nil

Description

Returns the path of a script.

Arguments

x_script

Script handle

Value Returned

t_path

The script path is returned.

nil

The script path is not returned.

Examples

Returns the path to the script with the handle 1045.

axlGetScriptPath(1045)
=> "myData/scripts"
