maeGetFaultSamplingOptions

maeGetFaultSamplingOptions(
[ ?session t_sessionName ] 
)
=> l_samplingOptions / nil

Description

Returns the sampling options set for fault simulation.

Arguments

?session t_sessionName

Name of a session

Default: Current session

Value Returned

l_samplingOptions

A disembodied property list of fault sampling options and their values.

nil

When the given session is not found.

Examples

The following example shows how this function returns fault sampling options: 

maeGetFaultSamplingOptions()
(nil enableSampling t samplingMethod "randomweighted" sampleRatio "50" seedValue "1" confidenceValue 97.0 )
