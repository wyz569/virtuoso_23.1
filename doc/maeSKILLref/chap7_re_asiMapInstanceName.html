asiMapInstanceName

asiMapInstanceName( 
t_dataDir
l_specifier
[ ?formatflag s_formatflag ]
) 
=> l_specifier

Description

Maps the hierarchical schematic instance name to the name in the netlist.

Arguments

t_dataDir

Specifies the data directory.

l_specifier

Lists the name of the instance to map.

?formatflag s_formatflag

Performs name mapping of transient simulation data available in SST2 format.

Set the value to "t" if this function is used in context of transient simulation data analysis and the transient simulation data is available in SST2 format.

Value Returned\

l_specifier

Returns the list containing the mapped name.

The list l_specifier passed as the argument is also destructively modified to contain the mapped name.
