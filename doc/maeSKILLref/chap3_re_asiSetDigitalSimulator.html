asiSetDigitalSimulator

asiSetDigitalSimulator( 
o_tool
s_digitalSimulator
)
=> s_digitalSimulator

Description

Sets the digital simulator.

Arguments

o_tool

Name of the tool.

s_digitalSimulator

Digital simulator name.

Value Returned

s_digitalSimulator

Digital simulator name that has been set.

Examples

asiSetDigitalSimulator( asiGetTool('Spectre) 'ams )
