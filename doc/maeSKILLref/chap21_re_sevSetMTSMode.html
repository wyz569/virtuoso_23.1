sevSetMTSMode

sevSetMTSMode(
o_session
g_mtsMode
[ g_noPopups ]
)
=> t / nil

Description

Enables or disables MTS mode.

Arguments

o_session

The simulation environment session object.

g_mtsMode

Enables or disables MTS mode.

g_noPopups

Controls whether the function pops up a warning message or not when the config view is in edit mode.

Value Returned

t

MTS mode is set successfully.

nil

MTS mode cannot be set because of an error.

Example

The following example sets MTS mode.

sess = (axlGetWindowSession (hiGetCurrentWindow))
sdb = (axlGetMainSetupDB sess)
(foreach testname (cadr (axlGetTests sdb) )
    (printf "Test %s\n" testname)
    sevsess = (axlGetToolSession sess testname)
    mtsmode = (sevMTSMode sevsess)
    (printf "\tMTS mode = %L\n" mtsmode)
    ;Example settng MTS mode t
    (unless mtsmode
     (sevSetMTSMode sevsess t)
 )
)
