artFlush

artFlush( 
x_handle 
) 
=> t / nil

Description

Flushes the file associated with the given handle.

Argument

x_handle

File handle returned from artOutfile.

Value Returned

t

Returns t if the file is flushed.

nil

Indicates an error.

Examples

artFlush( fHandle )
