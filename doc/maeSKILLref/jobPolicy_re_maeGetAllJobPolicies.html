maeGetAllJobPolicies

maeGetAllJobPolicies()
=> l_jobPolicyNames / nil

Description

Returns the job policies of all the tests in the specified session.

Arguments

None

Value Returned

l_jobPolicyNames

List of the job policy names is returned.

nil

List of the job policy names is not returned.

Examples

Returns all the job policies for the session fnxSession5.

maeGetAllJobPolicies()
=> ("Job Policy LPF" "Maestro Default" "Maestro1" "Netlisting Default")
