Cadence SKILL API Finder

Use the Cadence SKILL API Finder tool to display the description and syntax of the SKILL functions. To open the SKILL API Finder from the CIW, select Tools – SKILL Finder. The Cadence SKILL API Finder appears.

For information about using the SKILL API Finder, refer to the Using SKILL API Finder appendix in the Cadence SKILL IDE User Guide or click Help – Finder Help in the Finder.
