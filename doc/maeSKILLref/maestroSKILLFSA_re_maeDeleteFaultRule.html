maeDeleteFaultRule

maeDeleteFaultRule(
t_faultRuleName
[ ?session t_sessionName ]
)
=> t / nil

Description

Deletes the specified fault rule from the active fault setup.

Arguments

t_faultRuleName

Name of the fault rule to be deleted.

?session t_sessionName

Name of a session.

Default: Current session

Value Returned

t

When the fault rule is deleted successfully.

nil

In case of an error.

Example

The following example shows how to delete an existing fault rule named RB1:

maeDeleteFaultRule("RB1")
=> t
