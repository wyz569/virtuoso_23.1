axlDeleteJobPolicy

axlDeleteJobPolicy( 
t_jobPolicyName 
) 
=> t / nil

Description

Deletes the named job policy from the setup.

Arguments

t_jobPolicyName

Job policy name.

Value Returned

t

The named job policy is deleted.

nil

The named job policy is not deleted.

Examples

Delete the job policy, mypolicy:

axlDeleteJobPolicy( "mypolicy" )
=> t
