axlStopAll

axlStopAll( 
t_session 
) 
=> t / nil

Description

Stops all runs currently evaluating in the session.

Argument

t_session

Session name.

Value Returned

t

Stopped all runs in the current session.

nil

The command was unsuccessful.

Examples

sess=axlGetWindowSession(window(3))
=> "session0"
axlStopAll( sess) 
=> t
