asiInitVerilogEnvOption

asiInitVerilogEnvOption( 
o_tool 
)
=> t

Description

Initializes the base Verilog environment options.

Arguments

o_tool

Name of the Verilog tool.

Value Returned

t

The base Verilog environment options are initilized.

Examples

asiInitVerilogEnvOption(asiGetTool('spectre))
=> t
