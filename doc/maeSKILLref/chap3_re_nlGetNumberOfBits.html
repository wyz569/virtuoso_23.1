nlGetNumberOfBits

nlGetNumberOfBits( 
o_instance 
t_terminal 
) 
=> x_bits

Description

Returns the number of bits on the instance and terminal specified.

Arguments

o_instance

The instance object.

t_terminal

The schematic name of the terminal.

Value Returned

x_bits

A non-negative integer value representing the bit number.

Examples

nlGetNumberOfBits( inst "in")
