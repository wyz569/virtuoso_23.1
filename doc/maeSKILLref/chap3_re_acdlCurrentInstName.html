acdlCurrentInstName

acdlCurrentInstName(
)
=> t_mappedInstName

Description

Maps instance names during hierarchical or flat netlisting.

You can control casing of names during hierarchical netlisting using the instNameCase .cdsenv environment variable in following way:

  • Mapped names use mixed casing. This is the default setting. 
    auCdl.envOpts instNameCase  string "mixed" nil
  • Mapped names are in lower case. 
    auCdl.envOpts instNameCase  string "lower" nil
  • Mapped names are in upper case. 
    auCdl.envOpts instNameCase  string "upper" nil  ; give in upper case

Arguments

None

Value Returned

t_mappedInstName

Name of the mapped instance.

Examples

acdlCurrentInstName()
