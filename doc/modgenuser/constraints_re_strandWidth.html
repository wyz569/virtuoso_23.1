strandWidth

Definition

Specifies the required exact width of individual wire strands on the given layer.

Values
  • l_layerNames
    Specifies the layers to which the constraint must be applied.
  • x_strandWidth
    Specifies the width of individual wire strands on the given layers.
    Type: Integer

Parameters

None

Applies To
  • Trunk constraints
  • Twig constraints
  • Strap constraints

Example

'("strandWidth" "Metal1" 0.03)

Sets 0.03 user units as the required width of individual strands on layer Metal1.
