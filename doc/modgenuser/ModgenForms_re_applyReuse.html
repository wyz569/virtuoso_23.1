Apply Reuse Template Form

(Layout EXL and Higher Tiers) Use the Apply Reuse Template form to generate a new Modgen constraint using a Modgen template file.

Field Description

Device Group

Specifies the device group under which the required Modgen pattern file is located.

Pattern File

Specifies the name of the Modgen reuse template file.

Rows

Specifies the number of rows to be included in the new Modgen.

The default value is 1.
