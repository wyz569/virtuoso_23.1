What happens if a special cell is registered multiple times using both Liberty and a special cell definition?

To avoid conflict due to multiple registration of special cells, set the option Override if new definition found in the Liberty/Tech Files section of the Libraries tab. If this option is selected, the last definition is used else the first definition for the cell is used.
