Virtuoso Power Manager Task Assistant
Product Version IC23.1, June 2023

How can I ensure that the boundary ports do not relate to the supply set of the default domain or the top domain if power domain is explicitly specified or automatically created by Virtuoso Power Manager?

Select the Use anonymous supply for top level ports option on the Export tab to avoid getting the boundary ports automatically related to the default domain or the top domain, which is explicitly specified or automatically created by the Power Manager.
