Product Version IC23.1, June 2023

How do I choose the prefixes for supply sets and power domains?

By default, based on the selection of power and ground nets, the tool creates a supply set with prefix SS_. For example, if power and ground is Vdd_1v8 and Vss, the supply set created by default are SS__Vdd_1v8__Vss. For power domains, you can specify the prefix in the Domain Name Prefix field. Based on the name specified for the domain name prefix and the supply set chosen, the power domain is named. For example, for domain name prefix as PD and supply set name as SS__Vdd_1v8__Vss, the power domain is created PD__SS__Vdd_1v8__Vss.
