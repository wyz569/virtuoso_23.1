How do I use regular expressions to register supply nets?

If you do not know the specific net names that exist down the hierarchy and there are some naming conventions followed, you can specify such regular expressions for nets. For example, if you add vdd as a regular expression, such as iovdd, avddd1, any net name that contains this is considered as power net. Similarly, if GND is specified as a regular expression, names, such as GNDA, GNDD, and so on, are considered as ground nets. Also, if [v][V] [d][D] [d][D] are defined as regular expressions, all combinations are considered, such as vdd, VdD, VDD, Vdd, and so on.
