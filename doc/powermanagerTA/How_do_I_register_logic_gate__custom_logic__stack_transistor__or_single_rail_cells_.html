How do I register logic gate, custom logic, stack transistor, or single rail cells?

To register cells as devices:

  1. Use Filter in the Cells group box to search for a cell. The Filter option gives the flexibility to easily filter and sort the desired choice of cells to be registered.
  2. Select a cell or a set of cells from the Cells group box to register as devices.
  3. Select the device type in the list box for assigning the devices.
  4. Click  to assign.
