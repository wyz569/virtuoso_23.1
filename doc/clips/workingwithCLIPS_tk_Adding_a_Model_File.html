Adding a Model File

To add a model file, perform the following steps.

  1. Choose Tools – Setup Model Files. The Add/Edit Model Files form appears.
  2. Click Add to open the file browser and select model files.
    To remove an existing model file, select the file and click Remove.
  3. Click OK.
