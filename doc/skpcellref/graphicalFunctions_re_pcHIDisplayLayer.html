pcHIDisplayLayer

pcHIDisplayLayer()
=> t / nil

Description

Highlights a parameterized layer group in the current cellview window and displays Show Parameterized Layer text window with information about the highlighted group. When there is more than one parameterized layer group, click OK in the text window to view the next one.

Arguments

None

Value Returned

t

Highlights parameterized layer groups and displays the Show Parameterized Layer text window.

nil

There are no parameterized layer groups or the Show Parameterized Layer text window did not display.
