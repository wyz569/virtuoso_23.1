How do I customize the LVA checks?

The VRF Option form lets you specify the checks to be run on IO pins by the LVA checker. To customize LVA checks:

  1. Select Module – Options to display the VRF Options form.
    The     Check Locations section specifies the checks to be run by the LVA checker.

  2. Select Extra IOs to report IOs that are not present in the corresponding layout die.

  3. Select Missing IOs to report IOs that are present in the corresponding layout die.

  4. Check Mis-aligned IOs to check for IOs that do not follow the same alignment as their corresponding IOs in the layout die.

  5. Select Aligned IOs to check for all aligned IOs.

  6. Select Connectivity Missing IOs to check for IO pins that do not have valid connections.

  7. Select Connectivity Mismatch IOs to check for IO pins that have connectivity different from the corresponding pins in the layout die.

  8. Click OK to save the setting.
