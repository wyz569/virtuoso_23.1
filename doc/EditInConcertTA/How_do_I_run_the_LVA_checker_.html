How do I run the LVA checker?

Before running the layout vs abstract (LVA) checker, use the Virtuoso RF Options form to select the checks to be run. For more information, see How do I customize the LVA checks?

To run the LVA checker:

  1. Launch Edit-in-Concert mode from the board, package, module, or die layout of the required die instance.
  2. Move to the required tab.
  3. Choose Module – Layout Vs Abstract – Check IO Pad locations to start the LVA checker.

Violation markers are generated in the Annotation Browser to indicate connectivity mismatches, missing or extra IO pads, and misaligned IO pads.
