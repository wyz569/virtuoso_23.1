Behavioral Model

Symbol: bsource

Behavioral Source

This component can be used to model a resistor, inductor, capacitor, voltage or current source as a behavioral component. Using bsource you can express the value of a resistance, capacitance, voltage or current as a combination of node voltages, branch currents, time expression, and built-in expressions.

Command-line help

spectre -h bsource
