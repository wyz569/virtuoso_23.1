useRowHeight

APR.device.grid useRowHeight float float_number

Description

Specifies the horizontal height.

The default is 0.0.

GUI Equivalent

None

Examples

envGetVal("APR.device.grid" "useRowHeight")
envSetVal("APR.device.grid" "useRowHeight" 'float 1.0)
