metal_Width

APR.device.grid metal_Width float rowWidth

Description

Specifies the track width. The default value is 0.00.

GUI Equivalent

None

Examples

envGetVal("APR.device.grid" "metal_Width")
envSetVal("APR.device.grid" "metal_Width" 'float 2.00)
