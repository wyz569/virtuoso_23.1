Mesh Layer Setup Form

The Mesh Layer Setup form is used to control the pitch of the metal layer mesh.

Column Description

Layer

Displays the layer name on which mesh wire is to be created.

Control

Displays the selected control. The options are Skip Tracks, Mesh Count, or Term Align.

Value

The value depends on the selected control.

  • If Skip Tracks is selected, this is the value of the number of tracks that should be skipped between mesh wires.
  • If Mesh Count is selected, this value is the number of mesh wires that are created.
  • If Term Align is selected, it ensures that vertical mesh wires align with the source and drain pins of the devices.

Min tracks to skip

Specifies how many tracks should be skipped between mesh wires.

Environment Variable: route_meshMinSkipTracks, route_meshMinSkipTracksCount
