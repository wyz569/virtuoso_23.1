Congestion Analysis Task Assistant
Product Version IC23.1, June 2023

What is Congestion Analysis?

The Congestion Analysis assistant helps you perform comprehensive routing checks and ensure that accurate results are extracted and displayed. It is available in Layout EXL only and performs the following tasks:

  • Lets you view and analyze the routing density of a design
  • Helps you streamline the initial floorplanning through detailed placement and routing
  • Lets you tune global planning to improve routing convergence and perform congestion-based pin optimization
