Can I rename default global bias constraint name to a user-defined name?

You can rename the default global bias constraint name to a user-defined name.

To edit the name, click in the Global Bias text field and type the new name. 

You can add a net or group of nets selected in the Navigator assistant to the renamed global bias constraint. This is done using the + From Navigator button in the Global Bias Setup form.
