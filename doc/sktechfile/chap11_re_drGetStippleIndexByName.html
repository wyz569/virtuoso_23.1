drGetStippleIndexByName

drGetStippleIndexByName(
tx_display
t_stippleName
) 
=> x_stippleIndex / nil

Description

Reads virtual memory and returns the stipple index number.

Arguments

tx_display

The display device name or identifier.

t_stippleName

The stipple name.

Value Returned

x_stippleIndex

The stipple index number.

nil

The stipple does not exist for the specified display device.

Example

drGetStippleIndexByName("psb" "dots")
drGetStippleIndexByName(27832 "dots")

Reads virtual memory and returns the stipple index 3.
