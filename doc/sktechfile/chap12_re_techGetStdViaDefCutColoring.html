techGetStdViaDefCutColoring

techGetStdViaDefCutColoring(
d_viaDefID
)
=> t_cutColoring / nil

Description

(Virtuoso Advanced Node for Layout Standard) Returns the cut coloring pattern to use for the cut patterns created for this type of stdVia.

Arguments

d_viaDefID

The database identifier of the via definition.

Value Returned

t_cutColoring

The cut coloring pattern on the via definition.

nil

The command was unsuccessful.

Example

cutColoring = techGetStdViaDefCutColoring( viaDefId )
