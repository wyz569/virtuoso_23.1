techManagerOpenTechToolBox

techManagerOpenTechToolBox(
)
=> t / nil

Description

Opens the Technology Tool Box form. It is equivalent to the CIW Tools – Technology File Manager command.

Arguments

None

Value Returned

t

The command was successful.

nil

The command was unsuccessful.
