lmCloseLib

lmCloseLib 
=> t / nil

Description

Closes the previously opened working library.

Call lmOpenLib before starting any S/SLG command or SKILL function and call lmCloseLib when you are done if you are not using lmLoadData to load the library command file.

Arguments

None

Example

lmCloseLib

Closes a working library previously opened by lmOpenLib.
