hedRegUICustomFunc

hedRegUICustomFunc(
s_uifunc
) 
=> t / nil

Description

Registers a UI customization function that gets called when a new Hierarchy Editor window is created.

Arguments

s_uifunc

A UI customization function that accepts the hiWindowId value of the window being customized.

Value Returned

t

The customization function was registered.

nil

Registration failed.
