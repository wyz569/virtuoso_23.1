How to specify routing preferences for vertical trunks?

To define routing preferences for vertical trunks:

  1. Select Enable Vertical Trunks.
  2. Select Route Vertical Trunks Over Device to allow vertical routes to be generated over devices.
  3. Select the nets to be connected to the vertical trunks in the All Nets box and click Add to move them to the Trunk Nets box.
  4. Specify a space-separated list of vertical net widths for the selected trunk nets.
  5. Select a Vertical Trunk Layer Name to specify the layer on which vertical trunks are generated.
  6. Specify the Vertical Trunk Min Number Cuts for the vias that connect the twigs to other objects.
  7. Select a Vertical Trunk Side to specify the side along which vertical trunks are to be generated. The available options are: left, right, both, and auto (default).
  8. Specify the trunk offset by selecting the reference layer from the Vert. Track Ref. Layer list and specifying the offset of the first device.
