How to create MPP guard rings using the Array Assistant?

To create an Multipart Path (MPP) guard ring:

  1. Set Type to MPP on the Guard Ring tab.
  2. Select the Net to which the guard ring needs to be connected. 
  3. Select an existing MPP guard ring definition from the MPP drop-down list.
  4. Select the sides for inserting guard rings.
  5. Click Apply.

An MPP guard ring is created around the array.
