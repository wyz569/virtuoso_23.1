How to define Modgen placement settings?

On the Placement tab of the Array Assistant:

  1. Select a device group that contains the instances to add to the Modgen. If the instances are not part of any device group, select Generic Group.
  2. Select a Modgen reuse template file from the File list.
  3. Click Load to load settings from the file. 
  4. Set Pattern Preset to Current, Uniform, Interdigitated (by 1), Compact (abut all devices), or Custom (specify a base pattern).
  5. Set Active Rows to Best Fit for optimal placement of custom patterns, Specify to specify the number of active rows, or Allow Placer to change Aspect Ratio to let the placer adjust the aspect ratio. 
  6. Specify the sides to insert dummy rows and columns. For surround dummies, select both options. Transition inserts dummy columns after the specified number of active device columns.
  7. Specify the number of fins or fingers to be added to the dummy rows and columns. Select Match to match the fins or fingers to the neighboring device.
  8. Verify that the Modgen pattern displayed in the Pattern box matches your requirements. 
  9. Specify the Horizontal and Vertical spacing between Modgen devices and select a reference layer.
  10. Select Abut to abut all Modgen devices.
  11. Click OK.
