How to specify routing preferences for horizontal trunks?

To specify routing preferences for horizontal trunks:

  1. Select Route Over Device to generate horizontal routes over devices.
  2. Select the required channel nets in the All Nets box and click Add to move them to the Channel Nets box.
  3. Specify a space-separated list of net widths in the Net Widths field for the selected channel nets.
  4. Specify the spacing between trunks in the Trunk to Trunk Spacing field.
  5. Select a Trunk Layer Name on which horizontal trunks are to be generated.
  6. Select a Gate Twig Layer Name on which the twigs connected to the gate terminal are generated.
  7. Specify the gate twig width in the Gate Twig Width field.
  8. Select a layer from the S/D Twig Layer Name list for generating source and drain terminals.
  9. Specify the minimum number of cuts (Min Number Cuts) for the vias connecting the twigs to other objects.
  10. Specify the source and drain twig width values in the S/D Twig Width field.
  11. Select Trim Trunks to trim the ends of the horizontal trunks while routing.
  12. Select Share Horizontal Tracks to share the horizontal trunks that are on the same layer and have the same connectivity.
  13. Specify the trunk offset by specifying a reference and offset.
  14. Select one of the following trunk anchor points:
    • Top (default): Trunks are anchored to the top-left vertex of the device row.
    • Bottom: Trunks are anchored to the bottom-left vertex of the device row.
