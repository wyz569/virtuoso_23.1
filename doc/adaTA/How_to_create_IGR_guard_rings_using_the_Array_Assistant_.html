How to create IGR guard rings using the Array Assistant?

To create an Identical Guard Ring (IGR):

  1. Set Type to IGR on the Guard Ring tab.
  2. Select a Net connection for the guard ring.
  3. Select an IGR definition from the IGR list. The IGR definition provided by the PDK defines all aspects of the guard ring, which helps the Modgen instantiate the guard ring correctly.
  4. Select one of the following IGR types:
    • Surround: The IGR surrounds all the instances.
    • Surround+Strip: The IGR surrounds all the instances, and strips of guard ring instances are inserted between one or more Modgen rows.
    • Grid: The IGR surrounds every instance or group of abutted instances separately.
  5. Select the sides around the Modgen on which you want to add guard rings: Left, Right, Bottom, Top.
  6. Specify the number of fins in the Strip Width in Fins field.
  7. Specify the number of rows to be inserted between strips of IGRs in the Rows Between Strips field.
  8. Click Apply.

An IGR as per your definition is generated in the layout canvas.
