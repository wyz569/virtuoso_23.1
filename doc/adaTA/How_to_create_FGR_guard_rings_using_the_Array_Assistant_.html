How to create FGR guard rings using the Array Assistant?

To create a Fluid Guard Ring (FGR):

  1. Set Type to FGR on the Guard Ring tab. The Wrap tab of the Create Guard Ring form is displayed.

  2. Select an FGR device from the Device list. 

  3. Specify the number of via rows in the Contact Rows field.

  4. Specify the width of the guard ring in the Path Width field.

  5. Specify the Net Name for the container cellview.

  6. Select a shape for the wrap-around guard ring: Rectangular or Rectilinear.

  7. Select Wrap common to create a common guard ring around the selected objects.

  8. Select Place at Minimum Distance to use the minSpacing rule to compute the spacing between the guard ring and the object layer.

  9. Specify the required Contact Settings, Implant Layers, and Outer Rings.

  10. Click Apply.

An FGR as per your definition is generated in the layout canvas.
