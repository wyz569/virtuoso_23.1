Array Assistant Quick Help

The Array Assistant is a unified interface that allows you to quickly create and edit Modgens without invoking the Modgen Editor. The form integrates the key options from the Modgen Placement and Modgen Routing toolbars. Select the instances that you want to include in the Modgen and use the options in the Array Assistant to generate a placed and routed Modgen. The options in the form are organized in four tabs that let you specify the placement settings, guard ring options, and topology and routing settings. The fourth tab lets you load and save settings to Modgen template files.

The Array Assistant is accessible from both the Constraint Manager assistant and the Auto P&R assistant.
