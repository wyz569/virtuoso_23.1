What current estimation modes are available in interactive SDR?

You can control how the Create Wire command estimates the current in the edited pathSeg and via. The current estimation modes that are available in interactive SDR are as follows:

  • Auto
    Estimates the current automatically in the last section of the wire according to the connected objects and flightline targets. 
  • Sum Connected Pins Currents
    Estimates the current after adding the current of all the connected pins. 
  • Maintain Constant Current
    Estimates the current after inheriting the current from the previous wire. 
  • Nearest Island Current
    Estimates the current based on all the pins connected by the flightline.
