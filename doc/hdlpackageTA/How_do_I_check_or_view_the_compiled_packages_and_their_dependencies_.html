How do I check or view the compiled packages and their dependencies?

Click Package Checking Viewer to view the compiled packages and their dependencies. It opens the package checking viewer that has two tabs, Table view and Tree view.

  • Table view: Lists all the package files compiled and found in Default scratch directory. There are two kinds of package files listed: Cadence Built-In package files that need to be pre-compiled (for example, UVM and UPF) and user-defined package files (which could either be specified in xrunArgs file or in the makelib table).
  • Tree view: Lists the compiled module-package dependency relationship, that is, the root node always remains a module and the child nodes are the package files related to the module.
