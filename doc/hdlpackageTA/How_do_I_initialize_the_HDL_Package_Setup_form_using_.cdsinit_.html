Product Version IC23.1, November 2022

How do I initialize the HDL Package Setup form using .cdsinit?

You can initialize the form as shown in the code snippet below.

$ProjDir mentioned in the following example is a Linux setenv variable. This variable defines the Project Linux absolute path for the wrealPKGCompileXrunArgs.f file and tmpCompileScratch1 & source folders.

;;   Bindkeys and other Stuff
load( prependInstallPath( "samples/local/schBindKeys.il" ) )

;; LOAD THE ENVIRONMENT FILE TO BE USED FOR THIS PKG RAK.
;envLoadFile(".cdsenv")

printf( ".cdsinit,INITIALIZATION FOR HDL PKG STARTS \n" )
; ENABLE BY DEFAULT VISTUOSO HDL PKG SETUP
envSetVal("hdlPkg" "hdlPkgEnableHdlSetup" 'toggle  '(t))

; UVM PACKAGES
envSetVal("hdlPkg" "hdlPkgEnableUVM" 'toggle  '(t))
envSetVal("hdlPkg" "hdlPkgUVMVersion" 'string "1.2")

; UPF PACKAGE
; envSetVal("hdlPkg" "hdlPkgEnableUPF" 'toggle  '(t))

; BASIC LIST OF xrunArgsPKG.f
envSetVal( "hdlPkg" "hdlPkgXrunArgsFileTableList" 'string "( (t \"$ProjDir/wrealPKGCompileXrunArgs.f\") )" )
;envSetVal( "hdlPkg" "hdlPkgXrunArgsFileTableList" 'string "((t \"$ProjDir/constantMathPKGCompileXrunArgs.f\") (t \"$ProjDir/wrealPKGCompileXrunArgs.f\") )" )

; FOLDER ABSOLUTE PATH WHERE ARE STORED THE COMPILATION & LOG FILE
envSetVal( "hdlPkg" "hdlPkgProjDir" 'string "$ProjDir/tmpCompileScratch1")

; ADVANCED MODE
envSetVal("hdlPkg" "hdlPkgAdvancedMode" 'toggle  '(t))

; DEFINITAION OF THE LOGIC LIBRARY AND PACKAGE FILES TO BE COMPILED
;envSetVal("hdlPkg" "hdlPkgMakeLibTree" 'string "((t \"constantsLib\" \"$ProjDir/source/math_constants_pkg.sv\" \"\") (t \"mathLib\" \"$ProjDir/source/math_functions_pkg.sv\" \"\") )")
envSetVal("hdlPkg" "hdlPkgMakeLibTree" 'string "((t \"constantsLib\" \"$ProjDir/source/math_constants_pkg.sv\" \"\") (t \"mathLib\" \"$ProjDir/source/math_functions_pkg.sv\" \"\") (t \"wrealLib\" \"$ProjDir/source/wreal_pkg.sv\" \"\") )")

printf( ".cdsinit,INITIALIZATION FOR HDL PKG : DONE.\n" )

; envSetVal("asimenv.startup" "ProjDirectDir" 'string "./simulation")
; envSetVal("asimenv.startup" "simulator" 'string "ams")

printf( ".cdsinit, for AMSD  simulation, Initialization DONE.\n" )
