skTabulateSKILL

skTabulateSKILL( 
) 
=> t

Description

Brings up the SKILL Tabulator UI form.

The SKILL Tabulator is part of the SKILL Surveyor program. Users should access the SKILL Tabulator through SKILL Surveyor, part of the Conversion tool box.

Arguments

None

Value Returned

t

Always returns t.

Examples

skTabulateSKILL() 
=> t
