ilSlotBoundp

ilSlotBoundp( 
obj 
t_slotName 
) 
=> t / nil

Description

Checks if a named slot is bound (has been assigned a value) to an instance.

Arguments

obj

An instance of some class.

t_slotName

Slot name.

Value Returned

t

The slot is bound.

nil

The slot is unbound.
It throws an error if obj or t_slotName is invalid.

Examples

myObject->slotX = 20
ilSlotBoundp(myObject "slotX")=> t
