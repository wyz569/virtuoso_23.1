clear

clear( 
) 
=> t

Description

Clears all tracing and breakpoints.

This function undoes the effects of tracef, tracep, tracev, and breakpt.

Arguments

None

Value Returned

t

Always returns t.

Examples

Untraces all functions and variables and clears all breakpoints.

clear() 
=> t
