isClass

isClass( 
g_object 
) 
=> t / nil

Description:

Checks if the given object is a class object.

Arguments

g_object

Any SKILL object.

Value Returned

t

If the given object is a class object.

nil

When the object is not a class object.

Examples

isClass( classOf( 5 ) ) => t
isClass( findClass( 'Point ) ) => t
isClass( 'noClass ) => nil
