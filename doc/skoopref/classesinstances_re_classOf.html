classOf

classOf( 
g_object
) 
=> u_classObject

Description

Returns the class object of which the given object is an instance.

Arguments

g_object

Any SKILL object.

Value Returned

u_classObject

Class object of which the given object is an instance.

Examples

classOf( 5 )
=> class:fixnum
className( classOf( 5 ))
=> fixnum
