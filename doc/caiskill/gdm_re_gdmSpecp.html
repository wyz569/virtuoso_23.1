gdmSpecp

gdmSpecp(
g_object
)
=> t / nil

Description

Checks whether an object is of type gdmSpec, a user-defined type of SKILL object.

Arguments

g_object

The object you want to check.

Value Returned

t

g_object is a gdmSpec object.

nil

g_object is not a gdmSpec object.
