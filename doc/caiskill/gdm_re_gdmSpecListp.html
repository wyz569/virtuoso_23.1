gdmSpecListp

gdmSpecListp(
g_object
)
=> t / nil

Description

Checks whether an object is a gdmSpecList.

Arguments

g_object

The object you want to check.

Value Returned

t

g_object is a gdmSpecList.

nil

g_object is not a gdmSpecList.
