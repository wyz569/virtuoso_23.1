psQueryPaperSize

psQueryPaperSize(
t_plotterName
t_paperSize
)
=> l_dimensions

Description

Returns the dimensions of the specified paper size.

Arguments

t_plotterName

Name of the plotter.

l_paperSizes

The paper size to query for dimensions.

Value Returned

l_dimensions

Returns the dimensions of the given paper size in inches.
