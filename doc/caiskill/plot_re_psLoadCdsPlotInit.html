psLoadCdsPlotInit

psLoadCdsPlotInit( ) 
=> t / nil

Description

Indicates if the .cdsplotinit has successfully loaded or not.

Arguments

None

Value Returned

t

The .cdsplotinit successfully loaded.

nil

The .cdsplotinit did not load.
