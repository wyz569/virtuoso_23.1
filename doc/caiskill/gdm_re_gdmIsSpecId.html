gdmIsSpecId

gdmIsSpecId(
g_object
)
=> t / nil

Description

Checks whether an object is a valid gdmSpec object.

Arguments

g_object

The object you want to check.

Value Returned

t

g_object is a valid gdmSpec object.

nil

g_object is not a valid gdmSpec object.
