psQueryPlotters

psQueryPlotters( ) 
=> l_plotters

Description

Returns a list of plotter names from the loaded .cdsplotinit files.

Arguments

None

Value Returned

l_plotters

Returns a list of plotter names from all .cdsplotinit files that are loaded.
