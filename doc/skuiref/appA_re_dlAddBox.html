dlAddBox

dlAddBox(
w_dlist 
x_penNumber 
l_point1 
l_point2 
[ s_tagSymbol ]
) 
=> t / nil

Description

Adds a box to the display list.

Arguments

w_dlist

Display list to add to.

x_penNumber

Pen number to use.

l_point1 and l_point2

Any two opposite corners of the box.

s_tagSymbol

Symbol to tag this object with.

Value Returned

t

Operation successful.

nil

Operation failed.
