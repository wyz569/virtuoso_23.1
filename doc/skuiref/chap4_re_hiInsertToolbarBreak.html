hiInsertToolbarBreak

hiInsertToolbarBreak(
w_windowID
s_toolbar 
) 
=> t / nil

Description

Inserts a toolbar break into the toolbar area of a window before the specified toolbar.

Arguments

w_windowID

The ID of the window that contains the toolbar.

s_toolbar

The name of the toolbar before which you want to insert the toolbar break.

Values Returned

t

The toolbar break was inserted in the window.

nil

The operation fails in case of an error.

Examples

Inserts a toolbar break before toolbar.

hiInsertToolbarBreak( window(1) toolbar )
=>t
