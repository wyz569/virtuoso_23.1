hiGetMultiClickTime

hiGetMultiClickTime( ) 
=> x_milliseconds

Description

Returns the number of milliseconds that must elapse before a mouse click is recognized as a single click rather than the second click of a double click.

Arguments

None

Value Returned

x_milliseconds

Number of milliseconds that must elapse before a mouse click as recognized as a click.
