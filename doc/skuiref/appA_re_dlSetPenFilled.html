dlSetPenFilled

dlSetPenFilled(
x_penNumber
[ w_penTable ]
)
=> t / nil

Description

Sets the fill style of a pen to SolidFill.

Arguments

x_penNumber

Pen number to be set.

w_penTable

Pen table to use. If you do not specify this argument, the default pen table is used.

Value Returned

t

The fill style was set.

nil

The fill style could not be set.
