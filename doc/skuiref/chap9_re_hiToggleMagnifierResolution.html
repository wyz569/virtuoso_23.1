hiToggleMagnifierResolution

hiToggleMagnifierResolution( )
=> t

Description

Turns on magnifier resolution in the canvas if the canvas is not currently using magnifier resolution. Otherwise turns off magnifier resolution.

Arguments

None.

Value Returned

t

Magnifier resolution successfully toggled on/off.
