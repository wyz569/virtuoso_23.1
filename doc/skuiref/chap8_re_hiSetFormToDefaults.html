hiSetFormToDefaults

hiSetFormToDefaults( 
r_form 
) 
=> t / nil

Description

Sets each field in the form to its default value, except the disclosure field value.

Callbacks for the fields is called if it results in the value of a field change.

Argument

r_form

Form to reset.

Value Returned

t

The default values are set.

nil

An error occurred. An error message is displayed.
