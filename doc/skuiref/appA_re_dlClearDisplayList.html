dlClearDisplayList

dlClearDisplayList(
w_displayList
) 
=> t / nil

Description

Removes all the objects from a display list.

Argument

w_displayList

Display list you are clearing of all objects.

Value Returned

t

Removes all the objects from the display list.

nil

Operation failed.
