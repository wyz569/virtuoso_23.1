hiToggleAnchorMagnifier

hiToggleAnchorMagnifier( )
=> t / nil

Description

Toggles the anchored state of the magnifier to anchored/un-anchored depending upon the current anchored state of the magnifier.

This function is enabled for windows where the root application type returned by deGetRootType is either "schematic", "maskLayout" or "schematicSymbol".

Argument

None.

Value Returned

t

Magnifier anchored state successfully changed.

nil

Action failed.
