hiGetProgress

hiGetProgress(
g_dboxID
)
=> x_currentProgress

Description

Gets the current progress (that is, the current step) from a progress dialog box.

Argument

g_dboxID

The ID of the progress dialog box.

Value Returned

x_currentProgress

The current progress (that is, the current step).
