hiSetProgressText

hiSetProgressText(
g_dboxID
t_text
)
=> t / nil

Description

Sets the text that is displayed above the progress bar in a progress dialog box.

Arguments

g_dboxID

The ID of the progress dialog box.

t_text

The text that is to be displayed above the progress bar.

Value Returned

t

The text was set.

nil

There was an error.
