Magnifier Functions

This section describes those functions that can be used for canvas magnification procedures at any time during a Virtuoso session.

These functions are:

  • hiGetMagnifierSize
  • hiIsMagnifierAnchored
  • hiIsMagnifierOpened
  • hiResizeMagnifier
  • hiScaleMagnifier
  • hiToggleAnchorMagnifier
  • hiToggleMagnifier
  • hiToggleMagnifierResolution
  • hiUpdateMagOptions
  • hiUseMagnifierResolution
  • hiUsingMagnifierResolution
  • hiZoomInMagnifier
  • hiZoomOutMagnifier

See Using the Magnifier in the Virtuoso Schematic Editor L User Guide.
