hiSetSplashIcon

hiSetSplashIcon(
g_icon
)
=> t / nil

Description

Sets a new icon for an application splash screen. The icon is displayed between the product name and the license information and always be set to a 48x48 size.

Argument

g_icon

The icon for the splash screen.

Value Returned

t

The icon was set.

nil

The icon was not set.
