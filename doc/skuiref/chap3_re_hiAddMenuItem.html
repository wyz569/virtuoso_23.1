hiAddMenuItem

hiAddMenuItem(
r_hiMenu 
r_menuItem
) 
=> t / nil

Description

Adds the menu item or action to the end of all instances of the specified hiMenu.

Arguments

r_hiMenu

Menu created with hiCreateMenu or hiCreatePulldownMenu or hiCreateSeparatorMenuItem.

r_menuItem

Menu item description returned by hiCreateMenuItem or hiCreateSliderMenuItem or action returned hiCreateAction.

Values Returned

t

The menu item was added.

nil

The menu item was not added. An error message is also returned.
