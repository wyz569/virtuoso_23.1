dlFitDlistOnDraw

dlFitDlistOnDraw(
w_displayList
x_widgetId
g_doFit 
) 
=> t / nil

Description

Specifies whether a display list should automatically fit within the specified widget.

Arguments

w_displayList

Display list.

x_widgetId

ID the display list is attached to.

g_doFit

The display list automatically fits in the widget.
