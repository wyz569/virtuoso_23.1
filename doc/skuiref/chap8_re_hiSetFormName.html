hiSetFormName

hiSetFormName( 
r_form 
t_newTitle 
) 
=> t

Description

Changes the form title, which is displayed in the window frame banner. The function can be called by its alias hiChangeFormTitle.

Arguments

r_form

Specifies the form.

t_newTitle

New title.

Value Returned

t

The title is changed.
