hiFormClose

hiFormClose(
r_form
)
=> t

Description

Equivalent to clicking the Close button on a form. This function is an alias for hiFormDone. The Close button is identical to the OK button, except that it triggers hiFormClose instead of hiFormDone.

Argument

r_form

Form handle.

Value Returned

t

The form was closed.
