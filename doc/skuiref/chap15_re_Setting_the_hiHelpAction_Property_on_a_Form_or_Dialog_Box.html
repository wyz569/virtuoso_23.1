Setting the hiHelpAction Property on a Form or Dialog Box

  • After creating a form or dialog box, add the hiHelpAction property to the symbol that represents the form or dialog box name. Use the following syntax:
    putprop( formOrDBoxSymbol propertyValue ’hiHelpAction)
    where formOrDBoxSymbol is the symbol that represents the form or dialog box name and propertyValue is the name (symbol or string) of the callback function that you provide.
    For example, if your form symbol is ’myForm and your callback function is myHelpCB(), you would do the following:
    putprop( ’myForm ’myHelpCB ’hiHelpAction)
    If you do not want your callback function to be passed the default arguments, you must specify the value of hiHelpAction as a string that contains parentheses.
