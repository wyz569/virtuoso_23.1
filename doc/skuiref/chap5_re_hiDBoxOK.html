hiDBoxOK

hiDBoxOK( 
g_dboxID 
[ g_dontUnmanage ] 
) 
=> t / nil

Description

Function logged when the OK, Yes or Done button is pressed in a dialog box. This function can also be called directly from SKILL to simulate pressing OK, Yes or Done.

Do not use the g_dontUnmanage optional argument, as it is only intended for special quick help dialog boxes that are currently unused.

Arguments

g_dboxID

ID of the dialog box.

g_dontUnmanage

Do not use this argument.

Values Returned

t

The OK, Yes, or Done button was selected or an hiDBoxOK call was made from SKILL.

nil

Returned in all other cases.
