dlEnableItem

dlEnableItem(
w_dlist 
x_penNumber 
s_symbolTag 
g_enableOrDisable 
) 
=> t / nil

Description

Allows you to turn on or off shapes within a display list that match the provided symbol. All shapes on pen x_penNumber are searched and checked to see if the symbolTag matches.

Arguments

w_dlist

Display list to add.

x_penNumber

Pen number to turn shapes on or off.

s_symbolTag

Symbol to match to all shapes on pen x_penNumber.

g_enableOrDisable

t enables the shape; nil disables the shape.
