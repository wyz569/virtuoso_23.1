swindowp

swindowp(
g_object
)
=> t / nil

Description

Checks whether the specified object is a valid session window.

Argument

g_object

The object that you want to check.

Value Returned

t

g_object is a valid session window.

nil

g_object is not a valid session window.

Examples

swindowp( myWindowID )
