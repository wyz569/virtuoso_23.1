hiTreeRemoveAllItems

hiTreeRemoveAllItems(
g_tree
)
=> t / nil

Description

Removes all the items from a tree. To remove a single item, use hiTreeRemoveItem. To remove more than one item, use hiTreeRemoveItems.

You can add the item again with hiTreeAppendItem or hiTreePrependItem.

Argument

g_tree

The tree from which you want to remove all items. This is the return value of hiCreateTree.

Values Returned

t

The items were removed from the tree.

nil

The items were not removed from the tree.
