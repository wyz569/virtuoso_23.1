hiUpdateMagOptions

hiUpdateMagOptions(
)
=> t / nil

Description

Update and display the Magnifier Options form with the magnifier values that are being used in the current window.

Argument

None (uses current window).

Value Returned

t

Magnifier Options form successfully displayed and updated to use magnifier values in the current window.

nil

Magnifier options form not updated.

Example

hiUpdateMagOptions()
