hiGetProgressTotalSteps

hiGetProgressTotalSteps(
g_dboxID
)
=> x_totalSteps

Description

Gets the total number of steps of the progress bar in a progress dialog box.

Argument

g_dboxID

The ID of the progress dialog box.

Value Returned

x_totalSteps

The total number of steps on the progress bar.
