Online Help Functions

This section describes the SKILL functions that support the help mechanism. While these functions are available for your use, you typically do not need to use them directly. They are described here because they are used by the default help system.

The following SKILL functions are described in this section:

  • hiHelp
  • hiLookupHRFtag
  • hiModalDBoxHelp
  • hiModelessDBoxHelp
  • hiObShowTag
