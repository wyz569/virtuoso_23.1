Closing the Library Path Editor

To close the library path editor,

  1. Choose File – Exit.
    If you have not saved changes, a dialog box asks you whether you want to save your changes.
  2. Do one of the following:
    • Click Yes to save your changes and exit.
    • Click No to discard your changes and exit.
    • Click Cancel to dismiss the form without exiting.
