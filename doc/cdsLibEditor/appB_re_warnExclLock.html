warnExclLock

cdsLibEditor.main warnExclLock boolean { t | nil }

Description

Disables the popup messages. The default value is nil.

GUI Equivalent

None

Examples

envGetVal("cdsLibEditor.main" "warnExclLock")
envSetVal("cdsLibEditor.main" "warnExclLock" 'boolean t)
