Submit Form

Use this form to submit files to the project design management repository.

Field Description

Files to Submit

Lets you select files to submit to the project design management repository.

Select All

Selects all files in the list for submitting.

Deselect All

Deselects all files in the list so that no files are submitted.

Description (Optional)

Lets you type a description of your design changes.

Submit Options

This section lets you modify the submit operation.

Request Name

Lets you assign a name to the integration request.

Use Options

Lets you specify the options you want to use for the submit operation.
