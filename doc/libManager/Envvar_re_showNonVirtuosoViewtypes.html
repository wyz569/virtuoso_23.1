showNonVirtuosoViewtypes

cdsLibManager.main showNonVirtuosoViewtypes boolean { t | nil }

Description

Displays non-Virtuoso view types for the selected cell.

The default is t.

GUI Equivalent

Command

File – Open – Browse – Library Browser

Form Field

Show Non-Virtuoso View Types

Examples

envGetVal("cdsLibManager.main" "showNonVirtuosoViewtypes")
envSetVal("cdsLibManager.main" "showNonVirtuosoViewtypes" 'boolean nil)
