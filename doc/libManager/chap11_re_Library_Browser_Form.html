Library Browser Form

This form saves the information you type and restores it each time you reopen the form or the Virtuoso session.

Field Description

Show Categories

Expands the form to display categories.

Show Non-Virtuoso View Types

Displays non-Virtuoso view types for the selected cell.

Library

Displays the names of the libraries specified in your cds.lib file.

Cell

Displays the names of the cells in the selected library.

View

Displays the names of the views in the selected cell.
