Creating a New Library Without Specifying a Technology File

If you do not plan to design layouts, you do not need a technology file.

To create a new library without specifying a technology file, do the following:

  1. On the Technology File for New Library form, select Do not need process information.
  2. Click OK.

Library Manager creates the specified new library.

If you are not using a technology file, the system automatically attaches the default technology file default.tf when you open a library in Virtuoso.
