Modify ‘propertyName’ Form

Use this form to modify the property values.

Field Description

Defaults

Clears all values in the fields and sets Type to int.

Name    

Displays the name of the property.

Type

Lets you change the property type. Depending on the type you choose, the remaining fields change to prompt you for values for the property.

Value    

Lets you change the default value for the property.

Minimum Value    

Lets you change the minimum value for the property.

Maximum Value    

Lets you change the maximum value for the property.

Used for int, float, and time property types. The View Property Editor form or Cell Property Editor form displays these minimum and maximum values next to the property name.

Possible Choices

Lets you change the possible property values. Used for string property type only. The View Property Editor form or the Cell Property Editor form creates a list box of these choices.
