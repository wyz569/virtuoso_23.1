Copy Library File Form

Use this form to copy a library file to a new location.

Field Description

From

This section displays the information of the library file you want to copy.

Library    

Displays the name of the library containing the file you want to copy. Defaults to the selected library.

File    

Displays the name of the file you want to copy.

To

This section displays the information of the destination library file.

Library    

Displays the library to which the library file is copied.

File

Displays the name of the copied file.
