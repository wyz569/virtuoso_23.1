useOptionsOn

cdsLibManager.ckOut useOptionsOn boolean { t | nil }

Description

Specifies whether check in options are enabled.

The default is nil.

GUI Equivalent

Command

Design Manager – Show File Status

Form Field

Check Out – Use Options

Examples

envGetVal("cdsLibManager.ckOut" "useOptionsOn")
envSetVal("cdsLibManager.ckOut" "useOptionsOn" 'boolean t)
