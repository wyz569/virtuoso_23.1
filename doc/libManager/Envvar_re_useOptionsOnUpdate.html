useOptionsOn

cdsLibManager.update useOptionsOn boolean { t | nil }

Description

Specifies whether update options are enabled.

The default is nil.

GUI Equivalent

None

Examples

envGetVal("cdsLibManager.update" "useOptionsOn")
envSetVal("cdsLibManager.update" "useOptionsOn" 'boolean t)
