Copy Cell File Form

Use this form to copy a cell file from one library to another library.

Field Description

From

This section displays the information of the cell file you want to copy.

Library

Displays the name of the library containing the cell file you want to copy. Defaults to the selected library.

Cell

Displays the name of the cell containing the file you want to copy.

File

Displays the name of the file you want to copy.

To

This section displays the information of the cell file to where it is copied.

Library

Displays the library to which the cell file is copied.

Cell

Displays the name of the copied cell.

File

Displays the name of the copied file.
