Opening the Library Browser Form

To open the Library Browser form manually:

  1. Click Browse on a form. For example, click Browse on the Open File form.
  2. The cells in the Cells list box of the Open File form are displayed in alphanumeric order.
  3. To use the same sort order as used in Library Manager, set designEditor.fileSpec.
  4. The Library Browser form appears.
