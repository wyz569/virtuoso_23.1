saveAsText

cdsLibManager.defaults saveAsText string "save_as_name"

Description

Specifies the default Save As name for the settings file. The default is .cdsenv.

GUI Equivalent

Command

File – Save Defaults

Form Field

Save Defaults File As

Examples

envGetVal("cdsLibManager.defaults" "saveAsText")
envSetVal("cdsLibManager.defaults" "saveAsText" 'string "defaults")
