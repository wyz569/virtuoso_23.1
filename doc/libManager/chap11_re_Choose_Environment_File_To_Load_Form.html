Choose Environment File To Load Form

Use this form to load an environment file.

Field Description

Look in

Shows the directory you are searching for an environment file.

File name

Displays the name of the environment file you want to load.

Files of type

Lets you choose one of the following file types to show in the list of files:

  • Cds Environment (*.cdsenv)
  • All Files (*)
