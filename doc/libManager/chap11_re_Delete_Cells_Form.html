Delete Cells Form

Use this form to delete cells from the design.

Field Description

Delete

Lists all the cells in the design that you want to delete.

Don’t Delete

Lists all the cells in the design that you do not want to delete.

Select

Lets you specify a filter string for selecting all matching cell names in a list box.

Options

This section lets you choose to delete items locally only or also from the design management repository.

Delete Local And Inactive From DM System

Deletes your local copy and the copy in the design management repository.

Delete Local Only

Deletes the specified items from your Library Manager but not from the design management repository.
