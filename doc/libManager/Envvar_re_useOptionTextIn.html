useOptionText 

cdsLibManager.ckIn useOptionText string "optionsFile"

Description

Specifies check in options. The default is "".

GUI Equivalent

Command

Design Manager – Show File Status

Form Field

Check In – Use Options

Examples

envGetVal("cdsLibManager.ckIn" "useOptionText")
envSetVal("cdsLibManager.ckIn" "useOptionText" 'string "myOptionsFile.txt")
