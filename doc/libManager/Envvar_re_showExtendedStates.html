showExtendedStates

cdsLibManager.displayOptions showExtendedStates boolean { t | nil }

Description

Allows DM tables to be shown in any viewing mode.

The default is t.

GUI Equivalent

Command

View – Display Options

Form Field

For Objects – View – Show extended states

Examples

envGetVal("cdsLibManager.displayOptions" "showExtendedStates")
envSetVal("cdsLibManager.displayOptions" "showExtendedStates" 'boolean nil)
