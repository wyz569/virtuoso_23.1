useOptionText

cdsLibManager.ckOut useOptionText string "optionsFile"

Description

Specifies check out options. The default is "".

GUI Equivalent

Command

Design Manager – Show File Status

Form Field

Check Out – Use Options

Examples

envGetVal("cdsLibManager.ckOut" "useOptionText")
envSetVal("cdsLibManager.ckOut" "useOptionText" 'string "myOptionsFile.txt")
