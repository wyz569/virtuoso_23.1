Renaming a Category Using Library Manager

To rename a category, follow these steps:

  1. Select the category you want to rename.
  2. Choose Edit – Categories – Rename.
    The Rename Category message prompt appears.
  3. Specify the new category name in the To Category text box.
  4. Click OK.

The Library Manager renames the category.
