Delete Library Views Form

The Delete Library Views form displays a list of the paths for views you chose to delete on the Delete By View form. The name of the library containing these views is displayed above the list.

  • Views with a selected check box to the left of the row are selected for deletion.
  • Views with a deselected check box does not get deleted.

You can select or deselect views by clicking the toggle button to the left of the row.

Field Description

Select All

Selects all views and selects the check box to the left of each row.

Deselect All

Deselects all views and selects all check boxes.

Options

This section lets you choose to delete items locally only or also from the design management repository.

Delete Local And Inactive From DM System

Deletes your local copy and the copy in the design management repository.

Delete Local Only

Deletes the specified items from your Library Manager but not from the design management repository.
