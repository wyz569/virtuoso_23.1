Design Management

Generic Design Management (GDM) is the design management layer that interfaces with the particular design management system you are using. The GDM layer allows applications to interface with different design management systems using a set of basic commands .

When using any design management system that interfaces with GDM, you can optimize the performance of check-in and check-out operations by setting the DD_GDM_OPTIMIZE environment variable to yes before you start Virtuoso.
