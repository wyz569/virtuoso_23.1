Copying a View to Another Library

To copy a view to another library, do the following:

  1. Open the Copy View form.
  2. In the To group box in the Library field, type or select a destination library name.
    You can type a new library name or select an existing library from the drop-down list.
  3. In the To group box in the Cell field, type a destination cell name.
    If you do not change the cell name in the To group box, the copied cellview displays the same cell name as the original cell.
  4. In the To group box in the View field, type a destination view name.
    If you do not change the view name in the To group box, the copied view displays the same name as the original view.
  5. Click OK.

The From cellview is copied to the To cellview in the To Library. If the destination library does not already exist, the New Library form appears,  where you can specify a location and design management option for the new library.
