Delete Libraries Form

Use this form to delete libraries from the design.

Field Description

Delete

Lists all the libraries in the design that you want to delete.

Don’t Delete

Lists all the libraries in the design that you do not want to delete.

Select

Lets you specify a filter string for selecting all matching library names in a list box.

Options

This section lets you choose to delete items locally only or also from the design management repository.

Delete Local And Inactive From DM System

Deletes your local copy and the copy in the design management repository.

Delete Local Only

Deletes the specified items from your Library Manager but not from the design management repository.
