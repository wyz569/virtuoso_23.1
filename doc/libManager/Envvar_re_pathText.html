pathText

cdsLibManager.newLib pathText string "path_string"

Description

Specifies a default path string for creating a new library.

The default is "".

GUI Equivalent

None

Examples

envGetVal("cdsLibManager.newLib" "pathText")
envSetVal("cdsLibManager.newLib" "pathText" 'string "mypath.txt")
