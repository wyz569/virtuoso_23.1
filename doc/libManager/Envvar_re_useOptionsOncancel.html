useOptionsOn

cdsLibManager.ckCancel useOptionsOn boolean { t | nil }

Description

Specifies whether checkout cancellation options are enabled.

The default is nil.

GUI Equivalent

Command

Design Manager – Cancel Check Out

Form Field

Use Options

Examples

envGetVal("cdsLibManager.ckCancel" "useOptionsOn")
envSetVal("cdsLibManager.ckCancel" "useOptionsOn" 'boolean t)
